# Bii o ṣe le firanṣẹ ati Gbigba Zcash ti o ni aabo (ZEC) pẹlu Brave Wallet: Itọsọna Asiri Igbesẹ-ni-Igbesọ kan

Brave Wallet jẹ ààbò, àpò owó crypto abínibí tí a kọ ní tààràtà sínú aṣàwákiri Brave - kò nílò ìmúgbòòrò kankan.


## Ìdí Tí Ìrànlọ́wọ́ fún ZEC Tí Wọ́n Ń Dáàbò Bo Fi Ṣe Pàtàkì


Zcash jẹ́ ọ̀kan lára àwọn owó-ìpamọ́ tí ó ń pèsè ìpamọ́ nípasẹ̀ ìsopọ̀ pẹ̀lú àwọn ìsòwò tí a fi ààbò bo.

Brave Wallet n ṣe atilẹyin fun Orchard shielded ZEC, ti o jẹ ki awọn iṣowo aladani laisi sọfitiwia ẹnikẹta.

Awọn olumulo le bayi firanṣẹ / gba ZEC laisi fifihan awọn iwontunwonsi tabi metadata - gbogbo laarin aṣawakiri Brave.

## Part 3: Receiving Shielded ZEC (Unified Address)

** Àwọn ìgbésẹ̀ tó yẹ kó o gbé:**

1. Tẹ ZEC ninu awọn ohun-ini rẹ

2. Yan "Gba"

3. Brave generates a Unified Address (UA) - one address that supports both shielded and transparent transactions

4. Ṣe àdàkọ tàbí ṣayẹwo kóòdì QR

5. Pín UA rẹ pẹ̀lú ẹni tí ó rán an

## Apá 4: Fifiranṣẹ ZEC Ààbò ní Ìkọ̀kọ̀

** Àwọn ìgbésẹ̀ tó yẹ kó o gbé:**

## Àwọn oríṣi Adirẹsi

Eyi ni alaye fun iru adirẹsi Zcash kọọkan:

**t-addr (t1...)** - Transparent addresses work like Bitcoin addresses, meaning transactions are public on the blockchain. They are easier to use with non-Zcash wallets but do not provide privacy.

**z-addr (zs...)** - Adirẹsi ti o ni aabo lo imọ-ẹrọ asiri Zcash (Sprout / Sapling) lati fi oluranlowo, olugba, ati iye pamọ. Awọn bọtini wiwo nikan tabi oluwa le ṣafihan awọn alaye.

**UA (u... or QR)** - Unified Addresses combine multiple address types (transparent, shielded, Orchard) into one. This allows maximum compatibility while preferring privacy where possible.

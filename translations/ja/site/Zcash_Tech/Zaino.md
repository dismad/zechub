# Zaino インデクサー

Zaino は、Zingo チームによって Rust 言語で開発されたインデクサーであり、lightwalletd を置き換えることを目的とし、zcashd の廃止プロジェクトを推進することを目指しています。

Zaino は、フルブロックチェーン履歴を必要としないウォレットやアプリケーションなどの軽量クライアント向けに必要な機能を提供するだけでなく、フルノードまたはフルウォレットにも対応しています。また、Zebra や zcashd のフルバリデーターによって管理される最終確定ブロックチェーンおよび非確定ベストチェーンとメモプールへのアクセスも可能にし、ブロックエクスプローラーのサポートも提供しています。

## 新しいインデクサーが必要な理由

主な理由は将来に向けて準備を整えるためです。Zcashd と lightwalletd は2016年に bitcoind のコードからフォークされ、C++で構築されました。これらのサービスを構築するために使用されたプラットフォームおよびコードは、年月とともに古くなり、スケーラビリティや保守性が難しくなっており、現代的な機能の実装も困難になっています。

Rust は現代的で堅牢かつ安全なプログラミング言語であり、Zcash が将来的な開発に備えることを可能にし、多くの新しい開発者を引きつけ、Zcash エコシステム周辺での豊富な新機能の構築を促進します。

なお、可能な限り後方互換性を保つことも Zaino の目標です。API とインターフェースを通じて採用時の摩擦を減らし、Zcash エコシステム全体が Zaino の改善点から恩恵を受けられるようにすることを目指しています。これにより、大きなリライトや学習曲線の必要性は最小限に抑えられます。

さらに、RPC アクセスと完全なクライアントライブラリを通じて、軽量クライアント機能をフルノードから分離することが可能になります。このようにして開発者は Zaino を統合し、軽量クライアントアプリケーションから直接チェーンデータにアクセスできるようになり、Zebra ノードの機密情報を安全かつ孤立させた状態で保つことができます。

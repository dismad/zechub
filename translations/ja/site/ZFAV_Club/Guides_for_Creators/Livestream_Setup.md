# ライブ配信のセットアップ

**必要なもの:**

- オーバーレイがなくHDMI出力のあるビデオカメラ
- 三脚
- HDMIからUSBへの動画キャプチャデバイス（2台）
- スライドやデモをキャプチャするためのHDMI（無線が最適）
- ミキサーからの音声入力（無線が最適）
- OBSや類似ソフトウェアを実行できるノートPC（有線LANが最適）

*このキットの目的は、品質と携帯性です。*

この機材の一部をより手に入りやすい代替品で置き換える方法もあります（例：カメラではなくスマートフォンを使うなど）が、それは別のWiki投稿で説明します。

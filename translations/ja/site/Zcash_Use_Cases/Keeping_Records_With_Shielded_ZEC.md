# シールドされたZECの記録の保持

## TL;DR

- シールドされた資金はプライバシーが保たれていますが、それでも正確で完全な財務記録を残すことができます。
- メモはあなたの帳簿における明細項目として機能し、請求書番号や支払いの目的などになります。
- 見てみる鍵（ビューイングキー）を使用することで、あなたまたは選んだ人物（例：会計士）が記録を確認できますが、公開されることはありません。
- 任意の期間における収入と支出の合計は、報告書や税務に必要なものです。
- このようなことはあなたのプライバシーを損なうものではありません。あなたが誰に何を見せるかを自分で決めることができるからです。

<br/>

## 対象者

- ZECで支払われるフリーランスおよび小規模事業主
- プライバシーを保ちながら記録が必要な人
- 会計士や税務のために記録を作成している人

<br/>

## 挑戦

プライバシーと記録の保持は、対立するもののように感じられるかもしれません。もし取引がシールドされている場合、金額やアドレスは公開されないため、どうやって適切な帳簿をつけるか、または会計士に収入を見せることができるのかという疑問が生じます。

Zcashではこれは誤ったトレードオフではありません。シールドされた取引はデフォルトで誰の目にも見えませんが、Zcashはあなたが必要とする人に自分の記録を公開するためのツールも提供しています。世界に対してはプライバシーを保ちつつ、会計士には開かれた状態にすることが可能です。

<br/>

## メモが帳簿になります

すべてのシールドされた（zからz）取引は暗号化された[memo](/using-zcash/memos)を含むことができます。記録のためにメモは、支払いの目的を書く場所です：請求書番号、顧客名、プロジェクトコード、または「3月の家賃」といった短いノートなど。

メモは取引とともに移動し、関係者だけが読むことができるので、あなたの帳簿におけるプライバシーのある明細項目になります。あなたまたはクライアントが支払いごとに明確なメモを含める習慣があれば、取引履歴は文脈のない金額のリストではなく、使いやすい帳簿になります。

簡単な習慣：クライアントと常にメモに請求書番号を入れることに合意しましょう。後で支払いを請求書と照合する際には、簡単にマッチングできます。

<br/>

## 自分自身の履歴の確認

記録をつけるために、自分の活動を見ることは必要です。ウォレットはシールドされた取引を復号化するための鍵を持っているので、ウォレットはあなたの全体像を見せることができます：日付、金額、受け取りと送金の履歴、および添付されているメモ。

これは一般の人々が見ることができない部分ですが、あなたにはデータの所有者として見える権利があります。年間終了時にではなく、定期的に履歴を確認することで記録の正確性を保ち、間違いを見つけやすくなります。

<br/>

## 会計士との記録共有

他人にシールドされた活動を見せる必要がある場合（例：会計士や監査人）、支出鍵を渡したり、何かを公開したりする必要はありません。[ビューイングキー](/zcash-tech/viewing-keys)を共有します。

フルビューイングキーは読み取り専用です。所有者はアドレスの入金および出金取引（金額やメモも含む）を見ることができるが、資金を動かすことはできません。これは会計士に渡すべき安全な方法です。彼らが必要な視点を得られ、あなたのお金はあなた自身の管理下にあり、世界の他の人々には何も見えません。

これは選択的な開示と呼ばれ、シールドされたZcashが正直な記帳に対してではなく、それを利用して実用的に機能する理由の一つです。

<br/>

## 期間ごとの合計

多くの報告書では、時間帯にわたる合計が必要になります：この四半期で受け取った金額、送金した金額、純資産状況など。あなた自身の履歴を確認できるため、任意の期間（月、四半期、年）ごとにこれらの数値を加算できます。

メモを一貫して記録しておくと、これにより簡単になります。なぜなら、日付や金額だけでなく、支払いが何のために行われたかで支払いをグループ化できるからです。

<br/>

## 税務に関する注意点

税法は国によって異なり、時間とともに変化するため、これは一般的な情報であり、税務の助言ではありません。多くの地域では、暗号資産を受け取ったり処分したりすることが課税上の影響を及ぼす可能性があり、受け取った金額、時期、その時点での価値について記録を残すことが求められることがあります。

良いニュースは、シールドされたZcashがこれらの義務を果たすことを妨げることはないということです。あなたは完全なプライバシーを持つ記録を保持し、税務当局が必要とする期間ごとの合計を作成し、ビューイングキーを使用して会計士や税務当局に開示できますが、あなたの活動は公開されることはありません。義務について確信がない場合は、自国で資格を持った専門家に相談してください。

<br/>

## 避けたい一般的な間違い

- メモを飛ばすことで、年間終了時に金額だけが残り文脈がなくなってしまう
- すべての用途に同じアドレスを再利用することで、クライアントや目的ごとの区別が難しくなる
- 税務シーズンまで記録を見直すのではなく、日々の記録をつける習慣がない
- 会計士が必要とするのは読み取り専用のビューイングキーだけなのに、支出鍵を共有してしまう

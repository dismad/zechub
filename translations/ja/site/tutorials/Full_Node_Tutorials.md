# フルノードチュートリアル

Zcash フルノードは、ネットワーク上の透明かつシールドされた取引を検証します。フルノードを実行することで、ネットワークの耐久性と安定性に貢献できます。また、フルノードを運用することにより、ユーザーにとって安全性やセキュリティが向上します。以下は、zcashd または zebrad フルノードのセットアップ/アップグレードに関する動画ガイドです。

詳しくは、[ガイド](/site/Guides/Full_Nodes)をご覧ください。

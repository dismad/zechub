# ZecHubの構築にご協力ください

Zcashコミュニティの一員であり、ZecHubの構築に貢献したい場合は、いくつか非常に役立つ方法があります。

ニュースレター用のリンクを収集し、それらをニュースレターのGitHubページに追加してください: -> [手順はこちら](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Wikiドキュメントの事実確認を行い、編集を提案したり、新しいページを作成したりする

以下のカテゴリでビデオコンテンツを作成してください:
```
* Zcashの説明ビデオ
* Zcashウォレットガイド/チュートリアル
* 第三者アプリケーションのデモ
```

Zcashおよびプライバシー生態系向けにポスター/グラフィック/アニメーションなどのデザインを制作

既存のWikiページの翻訳を行う

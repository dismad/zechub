# SNSデータの収集

現在では、SNSの影響を受けないという人はほぼいないほど、SNSは広く普及しています。使いやすさやメッセージの拡散における速さと効率性により人気を博しました。

しかし、この同じ便利さが、ユーザーとして私たちがSNSプラットフォームが私たちについて収集する情報量やそれらの情報をどう利用するか、あるいは悪意のある第三者がアクセスしてしまった場合に何ができるかなどについて無関心になる原因ともなっています。

次に、これらのプラットフォームが私たちについて収集するデータとそれらをどのように使用し、それがなぜ私たちのプライバシーだけでなく、個人または家族の安全にも脅威となる可能性があるのかを説明します。

Discord
-------

まずDiscordから始めなければなりません。なぜなら、私たちは「自宅」で使っているコミュニケーションアプリだからです。

Discordは、幅広いトピックについての大規模なコミュニティを管理するのに非常に役立ちますが、他の多くの企業と同様に、私たちの個人情報を活用して利益を得ているという点では罪深いと言えます。

### Discordが保存するデータ

プライバシーポリシーによると、Discordは以下の情報を取り扱います：

- テキストメッセージ（下書きを含む）、ボイスメッセージ、ライブ配信のカバー画像、追加した友人ややり取りしたユーザー、プレイしているゲーム、登録されているサーバーまたはコミュニティ、アプリを通じて行った購入または販売、コンテンツモデレーション時に取る決定。

動画やライブ配信の動画は保存していませんが、将来的にこの方針を変更する可能性があると明記しています。

また、自動的に以下の情報も収集します：接続しているIPアドレス、使用しているブラウザ、オペレーティングシステムなどです。

### その利用方法

彼らが私たちのデータをどのように使うかという点では、多くの場合、ユーザー体験の改善やパーソナライズされた体験の提供、コミュニティ基準の維持といった正当な理由で使用されています。しかし、サービスの宣伝目的のためにも使われたり、第三者と協力してサービスを宣伝したり、企業成長の分析のためにデータを共有することもあります。

また、裁判所からの命令によって情報を当局に提供する可能性があるとも主張しています。

Twitter
-------

Discordと同じように、Twitterもサーバー上に多くの情報を保存しています。ダイレクトメッセージ、フォローしているユーザー、やり取りしているユーザー（保存された投稿やお気に入り）、そのポリシーは第三者や当局とデータを共有することを含め、非常に似ています。

Meta
----

Facebook、Instagram、Threadsを含むMetaのアプリケーション群はさらに進んでおり、あなたが提供する情報の一覧は長すぎて詳細に説明するのは難しいほどです。それはあなたの投稿（音声メモや写真も含む）、デバイスの「カメラ」機能またはギャラリーで撮影した写真、見ている広告、その広告を閲覧している時間、また意図的に無視した広告などすべてが含まれます。メッセージのメタデータ（場所、日時、デバイス）も含まれています。さらに、各アプリ機能ごとに費やす時間を含み、法律があなたを守らない場合、宗教的信念や哲学的信念、性的指向、さらには労働組合への所属まで含まれます。

それだけでなく、彼らの有名なPixelを通じてMetaはFacebook以外でのあなたの行動も把握できます：訪問したウェブサイト、そのウェブサイトにどのくらい時間を費やすか、プレイしているゲーム、購入しているもの、さらには教育レベルなどのデータです。もちろん、技術的なデバイス情報（IPアドレス、オペレーティングシステム、ブラウザなど）も含まれます。Facebookにログインしていなくても同様です。

なぜ私が気にすべきなのか？
--------------------------

「私は隠すものがない」というのが、これらのプラットフォームが私たちについて収集するデータの重要性を軽視するためによく使われる議論です。しかし、あなたがしていることが違法でない場合でも、これらの企業があなたについて知っていることに対して懸念する理由は十分にあります。

強力な組織が政治活動家と対立し、その行動が「間違っている」と判断した場合、彼らは情報を要求して、この人物に関する情報を提供することにより、深刻な影響を及ぼす可能性があります。[これは彼らが迷わず提供します](https://gizmodo.com/twitter-saudi-arabia-x-jack-dorsey-areej-al-sadhan-abdu-1850805085)。

また、あなたの個人情報が、あなたに悪意をもって利用する可能性のある他人、企業、ハッカーの手に渡る明確な例もあります。例えば、身分盗用、アカウントへのアクセス取得、不要な広告の送信、脅迫、[あなたを操作すること](https://en.wikipedia.org/wiki/Facebook%E2%80%93Cambridge_Analytica_data_scandal)などです。

さらに、あなたのデータに対するコントロールを失う可能性もあります。あなたは自分のデータがどのように保存され、処理され、第三者と共有されているのか、または気持ちが変わったときに削除したり変更したりできるか分からないかもしれません。これは、信頼していたサービスプロバイダーが[あなたのデータを適切に保護していない場合](https://www.iotworldtoday.com/security/duolingo-data-breach-exposes-3-million-user-emails) [正しく保護していない場合](https://edition.cnn.com/2023/01/05/tech/twitter-data-email-addresses/index.html)にもリスクをもたらす可能性があります。

代替手段はあるか？
-----------------------

人気はそれほど高くありませんが、少なくとも最も人気のあるSNSの多くには代替手段があります。いくつか見てみましょう：

### Matrix：Discordよりもプライバシーに優れたもの

MatrixはDiscordやTelegramに最も近い代替ソフトウェアで、多くの機能を共有していますが、端対端暗号化などのプライバシー向上機能を追加しています。データビジネスを行わず、コードをオープンソースとして保有している点も特徴です。あなた自身のサーバーを運用する場合は完全に無料ですが、インターネット上での専用サーバーを購入したり、プロジェクトへの寄付を行うこともできます。クロスプラットフォームで動作し、DiscordやTelegramなどの他の人気アプリケーションと統合することも可能です。

### Mastodon

Twitterの最も有名な代替ソフトウェアの一つです（ただし唯一ではありません）。Mastodonは分散型であることが特徴で、誰でも共通の興味や特性を持つ人々のために独自のインスタンスまたは「サーバー」を立ち上げることができます。しかし、他の独立したインスタンスとやり取りすることも可能です。

Mastodonでは、各管理者が独自のルールを設定しており、データビジネスを行っていないため、ユーザーに関する情報、人口統計、好みなどの情報をあまり収集しておらず、第三者とも共有していません。

ただし、メッセージに端対端暗号化は実装されていないので、この点には注意が必要です。

### Signal

メッセージングクライアントのWikiで詳しく説明していますが、SignalはTelegramやWhatsAppよりもプライバシーとセキュリティに優れた代替ソフトウェアです。非常に魅力的で興味深い機能を提供しています。

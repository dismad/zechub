# Zcash アバランチ RedBridge

Zcash アバランチ RedBridge は、Zcash（ZEC）とアバランチ（AVAX）のブロックチェーン間で相互運用性を実現する分散型ブリッジです。このブリッジは、アバランチが提供する高スループット、低手数料、環境に優しいコンセンサスメカニズムを活用しながらも、Zcash のプライバシー重視の特徴を維持して ZEC をアバランチブロックチェーンへとスムーズに転送することを目的としています。

RedBridge は、クロスチェーン型分散金融（DeFi）、プライベート取引、流動性共有など幅広いユースケースをサポートしており、Zcash 持有者がアバランチエコシステムへのアクセスを拡大できるようにしています。このブリッジは、**ZavaX** と呼ばれる分散型ノードおよびオラクルによって運用されており、Zcash とアバランチ間での信頼性の高いデータ転送および価格検証を実現しています。

### 主な特徴

プライバシーを保つ相互運用性: Zcash ユーザーがアバランチ上の DeFi アプリケーションを使用する際もプライバシーを維持できるようにします。
分散型オラクル ZavaX: オラクルシステムを統合し、正確な ZEC/AVAX 価格データを提供することで、信頼性の高いクロスチェーン操作が可能になります。
拡張性と環境に優しい設計: アバランチのコンセンサスモデルを利用し、最小限の環境への影響で高速なトランザクション処理を実現します。
DeFi および DApps のサポート: Zcash 持有者はプライバシーを損なうことなくアバランチ上のさまざまな DeFi プラットフォームに参加できます。

### 技術的構成要素

**分散型オラクル ZavaX**
説明: オラクルはブリッジの重要な部分であり、クロスチェーン価格フィードを提供し、信頼性の高い ZEC から AVAX への変換を可能にします。
[オラクルへのリンク](https://zavax-oracle.red.dev)

**クロスチェーンブリッジコントラクト**
説明: Zcash アバランチブリッジをサポートするスマートコントラクトアーキテクチャで、ZEC の預金、変換および引き出し処理を行います。

**プライバシーレイヤー統合**
説明: ブリッジングプロセス全体を通して Zcash のプライバシー機能を維持し、プライベートなクロスチェーントランザクションが可能になります。

* 配信物:
2025年第1四半期に、チームは[red·bridge デモウェブサイト](https://redbridge-demo.red.dev/index.html)のリリースを発表しました。このウェブサイトでは、誰でもユーザー体験を試すことができ、フィードバックや改善提案を行うことができます。また、非技術的な人々にもプロジェクトを簡単に紹介する手段として機能します。

* チームは最終バージョンの red·bridge に Zebra を使用しました。テストするために、彼らのテストブロックチェーンである ZavaX オラクル（アバランチの Fuji テストネット上で動作）の3つのノードのうち2つをアップグレードし、最後のノードも成功裏にアップグレードされました。現在、[Zavax Oracle](https://zavax-oracle.red.dev/) は ZEBRA 上で動作しています。

* 2025年第1四半期には、red.bridge ウェブサイトが red、Dark、Light、Zebra の4つのビューを提供するようにコード化されました。初期バージョンでは red のみでした。

* また、チームは2025年12月に red·bridge L1 をアバランチメインネット上でライブ運用することを計画しています。最初は Zcash ブロックチェーンのオラクルとして機能し、その後間もなく Bitcoin も対象とします。この際、各リクエストにはガストークンとして 0.001 AVAX のコストが発生します。この構築により、アバランチ上の任意の L1 またはスマートコントラクトが分散型で Zcash や Bitcoin からデータを低コストでクエリできるようになります。

* 第2四半期には、チームは Avalanche Foundation に ACP-77（通称: Avalanche9000）というマイルストーンを提出し、red.bridge ガーディアンの運用をより早くかつ誰にとっても負担が少ないようにしました。最初はバリデーターが約2000 AVAX をステーキングする必要がありました。しかし、Avalanche9000コストにより、バリデーターは1 AVAX（月）で済むようになりました。また、このマイルストーンでは ZF の FROST 実装の計画も最終的に確定し、各ガーディアンに署名シェアを提供してブリッジウォレットのセキュアな分散制御が可能になります。

* 2026年第1四半期および第2四半期には、red.bridge は RBR トークン（かつて ZAX）を Zcash とアバランチコミュニティメンバーにエ因い配布します。red.dev の創設者によると、インセンティブ付きテストネットを開催し、ユーザーがブリッジのテストに協力することで RBR を獲得する機会を提供する予定です。

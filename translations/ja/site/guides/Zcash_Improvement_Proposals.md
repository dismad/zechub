# ZIPとは
Zcash Improvement Proposal（ZIP）は、Zcashコミュニティに情報を提供するか、Zcashまたはそのプロセスや環境の新しい機能を記述するための設計文書です。ZIPは、機能の簡潔な技術仕様とその理由を提供する必要があります。

ZIPは、新機能を提案する主要な手段であり、特定の問題についてコミュニティからの意見を集める手段であり、Zcashに組み込まれた設計決定を記録するための手段であることを意図しています。ZIPのオーナー（通常は著者）は、コミュニティ内で合意形成を行う責任があり、異議のある意見も文書化する必要があります。

ZIPはバージョン管理されたリポジトリ内のテキストファイルとして維持されているため、その変更履歴は機能提案の歴史的記録となります。

-> [zips.z.cash](https://zips.z.cash)


## ZIP編集者

どのZIPが提出されても、それがレビューのプロセスを経る必要があります。ZIP編集者の役割は、任意のZIPの形式と内容が用語、理由および技術的な変更を十分に説明していることを確認することです。

```markdown
* Jack Grigg, Kris Nuttycombe および Daira-Emma Hopwood（ECC所属）
* Conrado および Arya（ZF所属）
* Sam および Mark（SL所属）
```

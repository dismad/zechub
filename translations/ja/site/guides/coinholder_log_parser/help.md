# 選挙ログフィルタ

`./filterForElection.sh <logname> <election #>`    

指定された選挙番号に関連するエントリの sighash と ballet の高さを表示します

`./numberOfBallets.sh <logname> <election #>`      

選挙番号に基づいて、ballet の高さの最新エントリ（最も最近）のカウントを表示します

`./displayCurrentResults.sh`                      

ログファイルを指定して、すべての選挙の概要を出力します

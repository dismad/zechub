# Zcashマイニングガイド: パーソナルハードウェアでマイニングプールに参加する方法

## はじめに

Zcash（ZEC）は、Equihashというプロオブワークアルゴリズムを使用してマイニングを行うプライバシーを重視した暗号通貨です。Zcashのマイニングでは、計算能力を使って複雑な数学的問題を解き、トランザクションを検証し、ネットワークを保護することでZEC報酬を得ます。ネットワークの難易度が高いため、ほとんどのユーザーにとってソロマイニングは推奨されません。マイニングプールに参加して、自分のハッシュパワーを他の人と組み合わせることで、安定した報酬を得るのが最善です。

このガイドでは、パーソナルハードウェア（例：GPU付きのホームPCやエントリーレベルのASIC）を使用してZcashをマイニングする方法に焦点を当てています。注記として、GPUはまだZcashをマイニングできますが、ネットワーク難易度の高い2026年ではASICの方がはるかに効率的で利益が出やすいです。常にWhatToMine.comなどのツールを使って現在の収益性を確認してください。電気代やハードウェア価格、ZEC価値などによって実現可能性が変化します。マイニングは全員にとって必ずしも利益があるわけではありませんので、地元の規制とエネルギー料金（1kWhあたり$0.08未満を目指す）を調査してください。

## 必要条件

### ハードウェア
- **GPUマイニング（初心者向けにパーソナル設定が推奨されます）:**
  - NVIDIAまたはAMDのGPUで、少なくとも4GB VRAM（例：NVIDIA GTX 1070, RTX 3060; AMD RX 580以上）。
  - 対応するマザーボード、十分な電源ユニット（複数のGPUの場合、750W以上）、および過熱を防ぐための良好な冷却装置。
  - マルチGPU構成はハッシュレート向上に適しており（例：6枚のGPUで1-2 kSol/s達成可能）。
- **ASICマイニング（より効率的だがコストが高め）:**
  - Bitmain Antminer Z15（420 kSol/s）やInnosilicon A9（50 kSol/s）などのEquihash対応ASIC。
  - 音が大きく、熱を発生しやすく、電力消費も大きい（例：1500W以上）。専用のスペースに適しています。Bitmain.comや販売業者（Blockware Mining）など信頼できるソースから購入してください。
- **一般的な条件:** 安定したインターネット接続と、設定・監視用のコンピュータ。ASICがネットワークを支配しており（2026年では13 GSol/sの総ハッシュレート）、GPUマイニングは競争力が低くても趣味家にとっては可能。

### ソフトウェア
- **オペレーティングシステム:** Windows 10/11、Linux（Ubuntuが安定性に優れている）。
- **マイニングソフト:**
  - GPU用：lolMiner（AMD/NVIDIA対応）、GMiner、またはminiZ（NVIDIA専用）。公式GitHubリポジトリからダウンロード（例：github.com/Lolliedieb/lolMiner-releases）。
  - ASIC用：メーカーの組み込みファームウェア/ダッシュボードを使用（例：BitmainのWebインターフェース）。
- **ウォレット:** 报酬を受け取るためのZcashウォレット。推奨:
  - シールド（プライベート）: Zashi Wallet、Zingo（モバイル/デスクトップ）、YWallet（モバイル/デスクトップ）。
  - テラントランスパレント（簡単だがプライバシーが低い）: Edge Wallet、Zecwallet Lite。
  - [ウォレット](https://zechub.wiki/wallets)からダウンロード。プールがサポートしている場合、プライバシーのためシールドアドレス（'zs'で始まる）を生成してください。

### その他
- 電気: コストを計算してください。GPUは1枚あたり150〜300W、ASICは1000W以上。
- アンチウイルス: 設定中に無効にしてください。アンチウイルスがマイナーを脅威として認識する可能性があります。

## マイニングプールに参加するためのステップバイステップガイド

### ステップ1：Zcashウォレットの設定
1. 公式Zcashウェブサイトからウォレットをダウンロードしてインストールしてください。[ウォレット](https://zechub.wiki/wallets)。
2. 新しいウォレットを作成し、シードフレーズを安全にバックアップしてください。
3. 受け取りアドレスを生成してください（プライバシーのためシールドが推奨されます）。記録してください。例：`zs1exampleaddress...`。
4. 透明アドレス（'t'で始まる）を使用する場合、簡単ですがプライバシーは低くなります。

### ステップ2：ハードウェアの準備
- **GPU用:**
  1. GPUをPCに取り付け、ドライバーを更新してください（NVIDIA: GeForce Experience; AMD: Radeon Software）。
  2. 経験がある場合はオーバークロックしてください（MSI Afterburnerを使用し、安定性を保つためにコアクロックは+100〜200、メモリは-500に設定）。
- **ASIC用:**
  1. ASICを電源とイーサネットに接続します。
  2. Advanced IP Scannerやメーカーのアプリを使用してIPアドレスを見つけます。
  3. Webダッシュボードにアクセスしてください（例：ブラウザでIPを入力し、Bitmainの場合デフォルトログインはroot/root）。

**警告:** 適切な換気を確保してください。マイニングは熱を発生します。まずは小規模からテストしてください。

### ステップ3：マイニングプールの選択と参加
マイニングプールは、あなたの寄与したハッシュレートに基づいて作業を分配し、報酬を共有します。料金（0〜2%）、支払い最低額（0.01〜0.1 ZEC）、場所（低遅延）、信頼性に基づいて選択してください。

**推奨プール（ハッシュレート、料金、レビューに基づく）:**
- **2Miners (zec.2miners.com)**: 1%手数料、PPLNS支払い、GPU/ASIC/NiceHash対応。高ハッシュレート（~1.17 GSol/s）、信頼性の高いサーバー。
- **F2Pool (zec.f2pool.com)**: 2%手数料、PPS+支払い、マルチコインサポート。大規模プール（~2.57 GSol/s）。
- **ViaBTC (zec.viabtc.com)**: 2%手数料（PPS+）、使いやすいダッシュボード、グローバルサーバー。
- **AntPool (zec.antpool.com)**: 1%手数料、Bitmainから、ASIC向け（~494 MSol/s）。
- その他：Kryptex Pool、Luxor（リアルタイム統計はpoolwatch.io/coin/zcashで確認してください）。

1. プールのウェブサイトにアクセスし、アカウントを作成します（メールまたは登録不要のものもあります）。
2. 設定でZcashウォレットアドレスを追加して支払いを受け取る準備をします。
3. プールのストラトムサーバー（例：zec.2miners.com:1010）とポートをメモしてください。

### ステップ4：マイニングソフトウェアのインストールと設定
- **GPU用（例：Windows/LinuxでのlolMiner）:**
  1. GitHubから最新バージョン（例：1.88）のlolMinerをダウンロードします。
  2. フォルダに解凍します。
  3. 設定付きのバッチファイル（start.bat）を作成します:
     ```
     lolMiner.exe --coin ZEC --pool zec.2miners.com:1010 --user YOUR_WALLET_ADDRESS.WORKER_NAME --pass x
     ```
     - `YOUR_WALLET_ADDRESS`をZECアドレスに置き換えます。
     - `WORKER_NAME`: ライドの名前（例：Rig1）。
     - EUサーバーを使用する場合: eu.zec.2miners.com:1010。
  4. バッチファイルを実行します。プールに接続しマイニングを開始します。
- **ASIC用（例：Bitmain Antminer）:**
  1. Webダッシュボードにログインします。
  2. マイナー設定に移動します。
  3. プールの詳細を追加:
     - URL: stratum+tcp://zec.2miners.com:1010
     - ユーザー名: YOUR_WALLET_ADDRESS.WORKER_NAME
     - パスワード: x（または空）。
  4. 保存し、マイナーを再起動します。
- **他のソフトウェア（例：GMiner）:**
  ```
  miner.exe --algo 125_4 --server zec.2miners.com:1010 --user YOUR_WALLET_ADDRESS.WORKER_NAME --pass x
  ```

**テスト:** 10〜15分間実行し、コンソールで承認されたシェアとハッシュレートを確認してください。

### ステップ5：マイニングの開始と監視
1. マイナーを起動します。プールに接続してシェアを送信し始めます。
2. 監視方法:
   - プールダッシュボード: ウォレットアドレスを入力してハッシュレート、未払い残高、統計を確認します。
   - ソフトウェアコンソール: エラー、温度（80度以下に保つ）を監視します。
   - ツール: HiveOSやSimpleMining OSを使用してリモートライド管理を行います。
3. 支払い: 多くのプールは最小限に達した場合自動で支払われます（例：0.05 ZEC）。プールルールを確認してください。

## ヒントとベストプラクティス
- **収益性:** whattomine.com/coins/166-zec-equihashなどの計算機を使用してください。例：RTX 3060（~300 Sol/s）は、ZEC価格が$50の場合、1日あたり約0.001 ZECを稼ぎ、電気代で約$0.50を差し引きます。
- **プライバシー:** シールドプールを使用可能であれば使用してください。アドレスの再利用は避けてください。
- **セキュリティ:** 強いパスワードを使用し、プール/ウォレットに2FAを有効にしてください。秘密鍵を他人と共有しないでください。
- **トラブルシューティング:** シェアが送信されない場合は、ファイアウォール、アンチウイルス、または設定ミスを確認してください。forum.zcashcommunity.comやReddit r/zecなどのフォーラムに参加してください。
- **代替案:** 無理な場合、クラウドマイニングや他のコインのステーキングを検討してください。
- **環境への注意:** マイニングはエネルギーを消費します。可能な限り再生可能エネルギーを使用してください。
- **アップデート:** Zcashは進化し続ける可能性があります（例：PoSへの移行）。z.cashで最新情報を確認してください。

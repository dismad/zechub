## zcash.me

![zcash.me ロゴ](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### 概要

**zcash.me** は、誰でも簡単にZcashユーザーにZECを送金できるプロフィールディレクトリおよび発見プラットフォームです。これは「ZcashのLinktree」として機能し、単一のリンクであなたのシールドされたアドレスとソーシャル存在感を共有する簡単な方法です。

| 項目              | 詳細 |
|-------------------|------|
| **ウェブサイト**       | [https://zcash.me/](https://zcash.me/) |
| **ツイッター**        | [@zcashme](https://x.com/zcashme) |
| **プロフィール形式**   | `zcash.me/[username]` |
| **状態**           | 運営中（開発中の機能あり） |

### 主な特徴

#### 現在利用可能
- **カスタムプロフィールページ** - ユーザー名、バイオ、アバター、所在地を含むパーソナライズされたページ
- **シールドアドレスの表示** - 一クリックでコピー可能な統合アドレス
- **ソーシャルリンクの統合** - X（旧ツイッター）、LinkedIn、Telegram、カスタムウェブサイト
- **認証システム** - コミュニティへの信頼を示す検証バッジ
- **リファラルランキング** - トップリファラーおよびコミュニティリーダーのトラッキング
- **アクティビティステータス** - 最後にアクティブになった時間の表示

#### 今後提供予定
| フィーチャー       | 説明 |
|------------------|------|
| 地図機能         | 周辺のZcashユーザーを検索可能 |
| プラットフォーム統計 | 成長メトリクスの表示 |
| コミュニティ投票   | エコシステムに関するトピックで投票可能 |
| 新聞 & 公式発表    | 公式的なアップデート |
| フォーラム        | コミュニティディスカッション |
| モバイルアプリ     | 完全なモバイル体験 |

### スタートする方法
1. [https://zcash.me/](https://zcash.me/) にアクセスし、**Join**をクリック
2. 独自のユーザー名を選択し、プロフィール写真と紹介文を追加
3. Zcash統合アドレス（`u1...`）を貼り付け
4. ソーシャルアカウントを接続
5. リンクを共有: `zcash.me/[yourusername]`

## TipZ

![TipZロゴ](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### 概要

**TipZ** は、クリエイターおよびサイファーパンクのために設計された **プライバシー保護のチッププロトコル**です。ファンが任意の主要な暗号資産でチップを送信すると、自動的に **シールドされたZEC**として変換され、送金されます。

| 項目              | 詳細 |
|-------------------|------|
| **ウェブサイト**       | [https://tipz.cash/](https://tipz.cash/) |
| **ツイッター**        | [@tipz_cash](https://x.com/tipz_cash) |
| **プロフィール形式**   | `tipz.cash/@[username]` |
| **状態**           | 運営中 |

> *"tipz.cashは、クリエイターやサイファーパンク向けのプライバシー保護チッププロトコルです。"*

### クリエイターにとってのプライバシーの重要性

#### 財務的自由
- プラットフォーム手数料0%
- 非信託および自己管理
- 中間業者なし

#### プライバシーファースト設計
- すべてのチップがシールドされたZECとして到着
- 秘密保持されたウォレット残高
- チップ付き暗号化メッセージ
- クリエイターの完全な匿名オプション

#### クロスチェーン支払い
| 貨幣 | ネットワーク |
|------|-------------|
| ETH  | Ethereum    |
| USDC | 多くのチェーン |
| SOL  | Solana      |
| ZEC  | 直接Zcash   |

#### クリエイター保護
- デプラットフォーミングリスクなし
- 記載禁止に強い
- KYC不要

### その他のリソース

```
TipZ 支払いフロー

ファン -> tipz.cashプロトコル -> クリエイターのウォレット（シールドされたZEC）

ファンが金額とメッセージを選択 -> プロトコルが自動変換 -> クリエイターがプライバシー保護されたシールドZECを受け取る
```

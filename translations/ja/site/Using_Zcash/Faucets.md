# Faucet（無料配布サービス）

<img width="2138" height="878" alt="Screenshot_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />

Faucet（無料配布サービス）とは、ユーザーが無料で少量の暗号資産を受け取れるサービスです。開発者はこのサービスを使って暗号資産を試験的に利用し、ユーザーは購入せずに少量の暗号資産を獲得できます。通常、一定時間内にユーザーが受け取ることができる暗号資産の量には制限があります。

メインネット: <a href="https://zecfaucet.com">zecfaucet</a>

テストネット: <a href="https://testnet.zecfaucet.com">testnet.zecfaucet.com</a>

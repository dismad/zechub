# Zcashネットワークアップグレード 

数年にわたり、Zcashはプロトコルに対して重要な変更や改善を導入してきました。今日は、これらのすべてのアップグレードについて見ていきます：

[OverWinter:](https://bitzecbzc.github.io/blog/overwinter/index.html) ブロック347500でアクティブ化され、2018年6月26日に採掘されました。OverWinterはZcashの初期リリース後の最初のネットワークアップグレードでした。OverWinterの主な目的は、将来的なネットワークアップグレードのためにプロトコルを強化することです。OverWinterの中心には、ネットワークアップグレード用の再プレイ保護、バージョニング、透明取引のパフォーマンス向上、および取引期限切れという新しい機能があります。

[Sapling:](https://coinbureau.com/analysis/zcash-sapling-upgrade/) ブロック419200でアクティブ化され、2018年10月29日に採掘されました。これはZcashネットワークにとって2つ目の主要な印象的なアップグレードであり、主にシールド取引用のzk-SNARKsの効率向上に焦点を当てています。Saplingリリースの際には、zk-SNARKsに関するいくつかの課題が顕在化していました。それはアップグレード性の問題から実装の複雑さ、信頼されたセットアップの要件までです。幸いにも、Saplingはzk-SNARK証明作成の効率を向上させることで、今日我々が享受しているように、この暗号通貨の潜在的な採用範囲を広げました。また、Saplingでは（パブリックパラメータ生成セレモニー）という重要な設定も構想されていました。これはZcashチームが改善しようとしていたものです。

[Zcash Blossom:](https://electriccoin.co/blog/blossom-upgrade-improves-speed-scalability-capacity/) ブロック653600でアクティブ化され、2019年12月11日に採掘されました。この重要なネットワークアップグレードは、ブロック時間の半分に短縮して約75秒にするなど、スケーラビリティとユーザー体験を改善することを目的としています。その結果？取引確認が速くなり、ネットワークの処理能力が倍増し、取引手数料は低コストになりました。Blossomアップグレードはすぐに示していますが、Zcashネットワークはセキュリティと信頼性の高い基準を維持しながらも、ネットワーク容量を拡大するための実用的なエンジニアリング決定を行うことができることです。

[HeartWood:](https://electriccoin.co/blog/introducing-heartwood/) ブロック903000でアクティブ化され、2020年7月16日に採掘されました。Heartwoodの唯一の目的は、シールドされたCoinbaseを通じてマイナーがシールドアドレスに報酬を受け取れるようにすることにより、より多くのサードパーティ統合とプライバシーを強化することです。さらに、Heartwoodはネットワークの分散性と相互運用性の向上によって裏付けられています。HeartwoodアップグレードではFlyclientも統合され、軽量クライアントが取引を効率的に検証できるようにし、スケーラビリティとサードパーティ統合を改善しました。また、ZIP 213として知られるシールドされたCoinbaseについても言及しておく価値があります。これはZcashコンセンサスルールを変更して、Coinbase資金がシールドされたSaplingアドレスにマイニングされるようにすることを目指しています。Saplingアップグレード以前は、シールドされたCoinbaseは実現不可能でした。なぜなら、シールド取引の作成には大量のメモリとCPUリソースが必要だったからです。

[Canopy:](https://youtu.be/R8O1SZMfESM?si=qoBL1dBp4E_af-eM) ブロック1046400でアクティブ化され、2020年11月18日に採掘されました。このアップグレードはElectric Coin Co (ECC)とZcash Foundationの双方によって裏付けられています。Canopyは創設者報酬の終了を示し、新しい資金調達メカニズム（Zcash開発基金）が導入され、Zcashエコシステムの継続的な資金提供を支える新しいガバナンスモデルも導入されました。Canopyでは次の4年間のために新しい開発基金が設立されます。80％の採掘報酬はマイナーに渡され、残り20％は新設された主要グランツ基金（8％）、Electric Coin Co（7％）、およびZcash Foundation（5％）に分割されます。「Canopy」という名前は、Zcashがプライバシーと分散性の原則を守りながらも持続可能で繁栄するエコシステムを作ることを目指していることを反映しています。

[NU5:](https://electriccoin.co/blog/nu5-proposed-features/) ブロック1687104でアクティブ化され、2022年5月31日に採掘されました。Zcashネットワークアップグレード5は、2016年の創設以来、この暗号通貨にとって重要なマイルストーンの幕開けを示しています。これはZcashにとって6つ目の主要なアップグレードであり、Orchardシールドプロトコル、統一アドレス、およびHalo証明システムが特徴です。Zcash NU5アップグレードは、信頼されたセットアップを排除し、プロトコルの基礎となる暗号技術のセキュリティを向上させるために構築されたzk-SNARK技術スタックの継続的な進化です。NU5はECCとZcash Foundationによっても支持されています。

[NU6:](https://zips.z.cash/zip-0253) NU6では、新しいZcash開発基金（ハイブリッドディレイド開発基金が非直接資金調達モデルへ移行）を実装し、その後、一部の発行額を将来的な分散型グランツ資金に備えるためのロックボックスを設立します。これらの資金の放出は、将来Zcashコミュニティによって決定されるメカニズムのみで統治されます。NU6の使命は、ブロック補助金を削減し、ロックボックスメカニズムを通じて分散型資金調達モデルを確立することにより、透明性を高めながらプライバシーを強化することです。

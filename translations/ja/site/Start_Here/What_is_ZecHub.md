# ZecHubとは？
---

ZecHubは、Zcash向けの分散型教育ハブです。[ZecHub](http://zechub.xyz)の目標は、コミュニティメンバーがZcashエコシステムをサポートするコンテンツの作成、検証、促進に協力できる教育プラットフォームを提供することです。いくつかのメカニズムを通じてこれを実現しています。

1. [チュートリアル](https://www.youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O)
2. [Wikiドキュメント](https://zechub.wiki/)
3. [コミュニティメンバーへのインタビューを含むポッドキャスト](https://www.youtube.com/playlist?list=PL6_epn0lASLHlNCMtUErX8UfaJK6N9K5O).
4. [週刊Zcashニュースレター。](https://zechub.substack.com/)

チュートリアル、短いブログ記事、ニュースレターはGitHub上でオープンソース化されており、コミュニティの誰でも貢献できます。これにより、コンテンツが正確であることを保証し、チャネル自体に単一障害点がないようにします。

Zcashに初めて触れる場合は、[このガイド](/site/Start_Here/New_User_Guide)をご利用ください。****

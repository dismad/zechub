# Zcashの基礎知識

## Zcashの経済構造とは？

Zcashの通貨供給量は、ビットコインと同様に2100万ZECという固定された上限があります。Zcashブロックチェーンにおいて、75秒ごとに新しい[block](https://zcash.readthedocs.io/en/latest/rtd_pages/glossary.html#:~:text=Block,mempool%20in%20an%20unconfirmed%20state.)が採掘され、3.125 ZECのブロック報酬が発行されます。このブロック報酬は、マイナーと[Zcash開発基金](https://zips.z.cash/zip-1014)に分配されます。

ブロック報酬の量は約4年ごとに半分になり、すべての2100万ZECが流通するまでこのプロセスが続きます。Zcashのインフレ率はほぼビットコインと一致します。新しいコインが作られるにつれてインフレ率は低下し、それぞれのハーフニング（報酬半減）ごとにその速度が大幅に減少することに注意が必要です。

# Zcashの基礎知識

## ZECの用途

#### 支払い

人々は、手数料が安く、安全に取引を行うためにZECを使用します。Shielded Zcashにより、取引は秘密に保たれ、必要であればアドレスや取引情報の共有も可能です。プライベートな「Z-Z」取引は公開ブロックチェーン上に表示され、取引が行われたことと手数料が支払われたことが確認できます。ただし、アドレス、取引金額、メモ欄はすべて暗号化されており、公開されることはありません。

#### 価値の保存

人々はまた、ZECを使って資産としての価値を硬貨のように、プライバシーを保護して保管することもできます。ZECの発行上限は常に2100万枚であり、この資産には固定された供給量があります。2100万枚目のZECが[採掘され流通に投入される](https://z.cash/mining-zcash/)と、その資産はインフレ防止効果を持つようになります。インフレ防止効果のある資産は、中央集権的な管理者が国家通貨の供給量を膨張させる可能性がある場合に、インフレに対する良いヘッジとなります。

#### メッセージ送信

人々は、Shielded Z-Z取引のメモ欄を使って完全に暗号化されたメッセージを送ることができます。メモを送るには、ユーザーが取引を行い、手数料（0.01ドル未満）を支払う必要があります。

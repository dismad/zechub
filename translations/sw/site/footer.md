Kuongeza maandishi haya kwa kila ukurasa kwenye tovuti dhana. 1) ni kwa kurasa kukamilika, 2) ni kwa ajili ya kurasa incomplete.

* Kama ungependa kuongeza, au kupendekeza mabadiliko ya ukurasa huu wiki, tafadhali kichwa kwa ZecHub github repo (kuongeza kiungo) na kuwasilisha ombi kuvuta.*

*Kama unataka kukamilisha ukurasa huu, tafadhali nenda kwenye repo ya github (ongeza kiungo), tengeneza ombi la kuvuta na, na ukamilishe ukurasa na nyaraka husika.*

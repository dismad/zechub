# Wachunguzi wa blockchain

## Explorer ni nini?

blockchain explorers ni zana tu ambayo kuruhusu moja ya kuona nini tayari kilichotokea juu ya blockchain. Wao kuchukua kwa ajili ya pembejeo: id shughuli, anwani, au block hashes, na visually pato nini kilichotukia.

## Orodha ya Zcash Block Explorer:

- [Zcash Block Explorer](https://mainnet.zcashexplorer.app/)

- [Kichumba cha kulala](https://blockchair.com)

- [3xpl](https://3xpl.com/zcash)

- [Bitquery](https://explorer.bitquery.io/zcash)

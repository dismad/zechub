## Tayari kutumia Zcash

Baada ya kusoma docs intro, na rasilimali zinazohusiana nao, kuangalia nje [Kutumia ZEC](https://zechub.wiki/guides/using-zec-privately#content) na [Tutorials](https://youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O&si=4vleWUbCGbNJu1-M) video kupata kuletwa kwa kutumia ZEC.

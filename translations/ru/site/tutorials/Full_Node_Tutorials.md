# Руководства по полным узлам

Полные узлы Zcash проверяют прозрачные и защищённые транзакции в сети. Запуская полный узел, вы вносите вклад в устойчивость и стабильность сети. Кроме того, работа с полным узлом предоставляет пользователю повышенные гарантии безопасности и защиты. Ниже приведены видеоруководства по установке/обновлению полных узлов zcashd или zebrad.

Пожалуйста, прочитайте наши [руководства](/site/Guides/Full_Nodes), чтобы получить более подробную информацию.

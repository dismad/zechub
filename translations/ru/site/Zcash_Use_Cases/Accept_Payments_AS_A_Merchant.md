### Шаг 1: Создайте адрес для оплаты

- Сгенерируйте защищённый (z) адрес
- Используйте его для получения платежей

### Шаг 2: Отобразите инструкции по оплате

Для каждой транзакции:
- Покажите свой z-адрес
- Укажите сумму платежа
- Опционально включите ссылку на мемо

### Шаг 3: Отслеживайте заказы

Используйте:
- Мемо (например, номер заказа)
- Или отслеживание вне цепочки (база данных, таблица)

Пример:
- Заказ #4821

### Шаг 4: Подтвердите оплату

- Наблюдайте за поступающими защищёнными транзакциями
- Сопоставьте мемо или сумму с заказом

### Шаг 5: Сохраняйте гигиену приватности

- Избегайте повторного использования адресов в публичном доступе, если это возможно
- Не раскрывайте историю транзакций
- Разделяйте кошельки для разных подразделений бизнеса при необходимости

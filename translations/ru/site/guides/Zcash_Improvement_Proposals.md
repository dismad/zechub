# Что такое ZIP
Предложение по улучшению Zcash (ZIP) — это проектный документ, предоставляющий информацию сообществу Zcash или описывающий новую функцию для Zcash, его процессов или среды. ZIP должен содержать краткую техническую спецификацию функции и обоснование этой функции.

Мы рассматриваем ZIP как основной механизм для предложения новых функций, сбора мнений сообщества по какому-либо вопросу и документирования проектных решений, принятых в Zcash. Владелец(ы) ZIP (обычно автор(ы)) несут ответственность за формирование консенсуса внутри сообщества и документирование особых мнений.

Поскольку ZIP поддерживаются в виде текстовых файлов в версионируемом репозитории, история их изменений является исторической записью предложения функции.

-> [zips.z.cash](https://zips.z.cash)


## Редакторы ZIP

Каждый поступающий ZIP должен пройти процесс рассмотрения. Задача редакторов ZIP — следить за тем, чтобы формат и содержание любого ZIP надлежащим образом описывали термины, обоснование и внесённые технические изменения. 

```markdown
* Jack Grigg, Kris Nuttycombe and Daira-Emma Hopwood associated with the ECC.
* Conrado, and Arya, associated with the ZF.
* Sam and Mark, associated with SL.
```

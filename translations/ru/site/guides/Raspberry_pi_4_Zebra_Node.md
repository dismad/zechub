# Руководство по запуску Zebra на Raspberry Pi 4

<img src="https://i.ibb.co/V3rjKwv/image-2023-11-28-172907488.png" alt="raspberry pi" width="300" height="300"/>

Запуск программного обеспечения узла Zebra на Raspberry Pi 4 позволяет вам участвовать в сети Zcash как независимый узел, совместимый с консенсусом. Это руководство поможет вам выполнить шаги по настройке и запуску Zebra на вашем Raspberry Pi 4.

## Предварительные требования

1. Raspberry Pi 4 (рекомендуется 2 ГБ ОЗУ или больше).

2. Карта MicroSD (рекомендуется 16 ГБ или больше) с установленной Raspberry Pi OS (Raspbian).

3. Стабильное интернет-соединение.

4. Клавиатура, мышь и монитор (для первоначальной настройки).

5. SSH-клиент (необязательно, для удалённого доступа).

## Установка

1. __Обновите систему__
   Откройте терминал или подключитесь к Raspberry Pi по SSH и убедитесь, что ваша система обновлена, выполнив:

   __sudo apt update__

   __sudo apt upgrade__

2. __Установите зависимости__
   Вам потребуется установить некоторые необходимые зависимости для сборки и запуска Zebra:

   __sudo apt install build-essential cmake git clang libssl-dev pkg-config__

3. __Клонируйте репозиторий Zebra__
   Откройте терминал и клонируйте репозиторий Zebra на ваш Raspberry Pi:

   __git clone https://github.com/ZcashFoundation/zebra.git__

   __cd zebra__

4. __Соберите Zebra__
   Чтобы собрать Zebra, используйте следующие команды:

   __cargo build --release__

   Этот процесс может занять некоторое время. Убедитесь, что ваш Raspberry Pi достаточно охлаждается, так как компиляция может вызывать нагрев.

5. __Конфигурация__
   Создайте файл конфигурации для Zebra. В качестве отправной точки можно использовать конфигурацию по умолчанию:

   __cp zcash.conf.example zcash.conf__

   Отредактируйте файл zcash.conf, чтобы настроить параметры вашего узла. Вы можете указать сеть, включить майнинг, настроить подключения к пирам и многое другое.

6. __Запустите Zebra__
   Теперь вы можете запустить Zebra с вашей пользовательской конфигурацией:

   __./target/release/zebrad -c zcash.conf__

   __git comment__ 

   Эта команда запустит узел Zebra, и он начнёт синхронизацию с блокчейном Zcash.

7. __Мониторинг__
   Вы можете отслеживать прогресс и состояние вашего узла Zebra, открыв веб-браузер и перейдя по адресу __http://127.0.0.1:8233/status__.

## Устранение неполадок

Если вы столкнётесь с какими-либо проблемами при сборке или запуске Zebra, ознакомьтесь с [документацией Zebra](https://doc.zebra.zfnd.org/docs/intro.html), где представлены советы по устранению неполадок и дополнительная информация.

Обязательно следите за тем, чтобы Raspberry Pi не перегревался, так как работа узла может вызывать нагрев. Возможно, вам стоит использовать систему охлаждения, например вентилятор или радиатор.

## Заключение

Следуя этому руководству, вы сможете успешно настроить и запустить Zebra на вашем Raspberry Pi 4. Теперь вы вносите вклад в сеть Zcash как независимый узел, помогая обеспечивать конфиденциальность транзакций Zcash.

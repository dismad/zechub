# Viewing Key

Экранированные адреса позволяют пользователям совершать транзакции, раскрывая как можно меньше информации в блокчейне Zcash. Что происходит, если вам нужно раскрыть конфиденциальную информацию об экранированной транзакции Zcash определённой стороне? Каждый экранированный адрес включает Viewing Key. Viewing Key были представлены в [ZIP 310](https://zips.z.cash/zip-0310) и добавлены в протокол в обновлении сети Sapling. Viewing Key — важнейшая часть Zcash, поскольку они позволяют пользователям выборочно раскрывать информацию о транзакциях.

### Зачем использовать Viewing Key?

Зачем пользователю вообще может понадобиться это делать? Из блога Electric Coin Co. по этой теме...

*- Биржа хочет определять, когда клиент вносит депозит в ZEC на экранированный адрес, при этом храня ключи **spend authority** на защищённом оборудовании. Биржа может сгенерировать incoming viewing key и загрузить его на подключённый к Интернету узел **detection**, в то время как ключ для расходования средств останется в более защищённой системе.*

*- Кастодиан должен предоставить аудиторам обзор своих активов в Zcash. Кастодиан может сгенерировать full viewing key для каждого из своих экранированных адресов и поделиться этим ключом с аудитором. Аудитор сможет проверить баланс этих адресов и просмотреть прошлую активность транзакций к этим адресам и от них.* 

*- Бирже может потребоваться провести проверки due diligence в отношении клиента, который делает депозиты с экранированного адреса. Биржа может запросить у клиента Viewing Key для его экранированного адреса и использовать его для проверки экранированной транзакционной активности клиента в рамках этих расширенных процедур due diligence.*

### Как найти свой Viewing Key

#### zcashd

* Выведите список всех известных адресов с помощью *./zcash-cli listaddresses*

* Затем выполните следующую команду либо для UA, либо для экранированных адресов Sapling

  ```bash
  ./zcash-cli z_exportviewingkey "<UA or Z address>"
  ```

#### Ywallet

* В правом верхнем углу выберите "Backup", подтвердите доступ на телефоне, затем просто скопируйте свой Viewing Key, который будет отображён.

### Как использовать свой Viewing Key

#### zcashd

* Используйте следующую команду с любым vkey или ukey: 

```bash
./zcash-cli z_importviewingkey "vkey/ukey" whenkeyisnew 30000
```

#### ywallet

* В правом верхнем углу выберите "Account", нажмите на "+" в правом нижнем углу, чтобы добавить и импортировать свой Viewing Key и добавить свою учётную запись только для чтения.

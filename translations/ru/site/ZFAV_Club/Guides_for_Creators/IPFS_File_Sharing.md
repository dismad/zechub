# P2P-Обмен файлами с использованием IPFS и браузера Brave

Мы можем использовать IPFS для обмена большими медиафайлами между нами в режиме Peer to-Peer.

Итак, какая проблема: шлюзы IPFS не работают с большими файлами, они будут тайм-аутить.

**Требования**: [IPFS Desktop](https://docs.ipfs.tech/install/ipfs-desktop/) + [Brave Browser](https://brave.com/)

tldr, Видео находится в конце.

Мы можем использовать это между нами в ZF AV Club для передачи больших медиафайлов и документировать это для вас всех, потому что это весело и очень круто!!

*ПЛЮС*, мы будем напрямую сотрудничать с командой IPFS в Protocol Labs для разработки этого использования случая.
Еще одно преимущество, возникшее из связей, сделанных на dWeb Camp!

Без установленного IPFS на принимающем устройстве и использование браузера, отличного от Brave: вам придется ждать, пока шлюз загрузит файл. *Это вернет ошибку 504 для больших файлов.*

Далее: как закрепить файлы других людей на вашем компьютере, чтобы у нас не было проблемы с единственной точкой отказа, когда исходный узел выходит из строя.

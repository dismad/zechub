# Генерация и перевод субтитров видео

Инструменты, используемые в этом руководстве:

1. CapCut
2. [Экспорт SRT из CapCut](https://www.biyaoyun.com/capcutsrt)
3. Текстовый редактор
4. [Онлайн-переводчик и редактор субтитров](https://www.syedgakbar.com/projects/dst)

Начните, импортировав окончательный вариант вашего видео в CapCut. Либо вы можете использовать CapCut для редактирования видео до продолжения этого процесса.

После того как окончательное видео загружено в CapCut, используйте [Экспорт SRT из CapCut](https://www.biyaoyun.com/capcutsrt), чтобы получить файл субтитров со временными метками из проекта CapCut.

Откройте файл SRT в текстовом редакторе и посмотрите видео, одновременно читая его в файле SRT. Вносите изменения в файл SRT при обнаружении ошибок. Используйте поиск и замену для **замены всех** случаев каждой ошибки.

Совет: смотрите видео в 2x, проверяя субтитры

После того как исходный текст был проверен и исправлен, загрузите файл SRT на [Онлайн-переводчик и редактор субтитров](https://www.syedgakbar.com/projects/dst), чтобы перевести файл SRT на любой другой язык за небольшую плату.

Переведенный файл SRT должен быть проверен с помощью программного обеспечения для редактирования видео кем-то, кто понимает переводный язык.

Переведенные файлы SRT иногда требуют корректировки временных меток, которые легко найти и исправить с помощью большинства программ для редактирования видео.

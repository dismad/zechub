# Настройка прямой трансляции

**Что вам понадобится:**

- Видеокамера с HDMI-выходом без наложений
- Трипод
- Устройство для захвата видео через HDMI-to-USB (2 штуки)
- HDMI для захвата слайдов презентации/демонстрации (лучше всего беспроводной)
- Аудио сигнал с микшера (лучше всего беспроводной)
- Ноутбук, на котором можно запустить OBS или подобное программное обеспечение (лучше всего с Ethernet-подключением)

*Цель этого комплекта — качество и мобильность.*

Существуют способы замены части этого оборудования более доступными решениями (например, использование телефона вместо камеры), которые будут обсуждаться в другом посте вики.

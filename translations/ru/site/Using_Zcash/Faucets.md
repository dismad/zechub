# Краны

<img width="2138" height="878" alt="Screenshot_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />


Краны — это сервисы, которые бесплатно выдают небольшие суммы криптовалюты. Их часто используют разработчики, чтобы экспериментировать с криптовалютой, а также пользователи, чтобы получать небольшие суммы криптовалюты без необходимости её покупать. Обычно они ограничивают количество криптовалюты, которое пользователь может получить за определённый период времени.

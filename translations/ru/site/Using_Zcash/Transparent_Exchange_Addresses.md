# Что такое TEX-адреса Zcash?

TEX-адреса Zcash представляют собой уникальный тип адреса для получения средств. Это акроним от «Transparent Exchange» address; такой адрес является **уникальной** кодировкой Unified-типа (bech32m) для одного Transparent-адреса p2pkh.

Его единственная цель — сообщить совместимому кошельку, что необходимо выполнить только Transparent-транзакцию (T -> T).

Логика следующая: при обнаружении TEX-адреса совместимый кошелёк декодирует его, чтобы получить содержащийся в нём Transparent-получатель. Затем кошелёк отправляет необходимые для транзакции средства из Shielded-пула на отдельный, контролируемый пользователем, временный Transparent-адрес (Z -> T). После этого он отправляет эти средства на декодированный Transparent-получатель TEX-адреса (T -> T).

Техническое предложение для TEX-адресов изложено в Zcash [ZIP 320](https://zips.z.cash/zip-0320), где определяется тип адреса, предназначенный исключительно для получения средств с Transparent-адресов.

Хотя TEX-адреса пока не получили широкого распространения, пользователям Zcash в конечном итоге может понадобиться их использовать.

## Когда мне нужен TEX-адрес

### TEX-адрес вам **нужен**, когда вы отправляете средства на Transparent-адрес с помощью кошелька, который не поддерживает прямую отправку на Transparent-адрес.
Некоторые кошельки просто не позволяют отправлять средства напрямую на Transparent-адрес, и **получатель может не предоставить эквивалентный TEX-адрес**. Поэтому иногда может потребоваться **преобразование** Transparent-адреса в TEX-адрес. Это можно сделать вручную, запустив эталонную реализацию, описанную в zip-320. Размещённый экземпляр **Transparent-to-TEX-Converter** доступен [ЗДЕСЬ](https://690e9524c66a3ecac5d54eff--jade-brioche-873777.netlify.app/).

### TEX-адрес вам нужен, когда вы отправляете средства на централизованную биржу, которая **ТРЕБУЕТ, чтобы эти средства поступали из Transparent-источника**.
В настоящее время [Binance](https://www.binance.com/) — единственная централизованная биржа, использующая TEX-адреса (и именно они стали основной причиной создания TEX).
TEX-адреса сообщают совместимому кошельку, что все средства, отправляемые на такой адрес, должны быть прозрачными, и исключают отправку любого shielded-значения на этот адрес.
Если такая биржа, как Binance, отклонит отправленную сумму, у неё будут необходимые средства, чтобы вернуть эту сумму обратно на адрес, откуда она поступила. Это также помогает таким организациям, как Binance, соблюдать законы и нормативные требования, установленные правительствами или другими регулирующими органами.

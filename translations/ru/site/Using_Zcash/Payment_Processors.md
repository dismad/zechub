# Платёжные процессоры Zcash

## CoinPayments
- **Тип поддержки**: Transparent   
- **Описание**: Глобальный платёжный шлюз для криптовалют, поддерживающий более 100 криптовалют, включая Zcash. Предлагает простую интеграцию для интернет-магазинов.  
- **URL**: [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="CoinPayments Logo" width="200"/>

---

## CipherPay
- **Тип поддержки**: Shielded   
- **Описание**: Принимайте Zcash за считанные минуты, без кастодиального хранения, без данных покупателей, без посредников.  
- **URL**: [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="CipherPay Logo" width="200"/>

---

## Flexa
- **Тип поддержки**: Transparent 
- **Описание**: Flexa — это процессор криптовалютных платежей, который позволяет продавцам принимать цифровые активы, включая Zcash (ZEC), для повседневных транзакций.  
- **URL**: [Flexa](https://flexa.co/)  
<img src="https://framerusercontent.com/images/7D7WwOyaY4PFVW3ZiBzGggx13Xo.svg" alt="Flexa Logo" width="200"/>

---

## BTCPay Server
- **Тип поддержки**: Shielded   
- **Описание**: BTCPay Server — это платёжный процессор криптовалют с открытым исходным кодом и возможностью самостоятельного хостинга.  
- **URL**: [BTCPay Server](https://github.com/btcpay-zcash/btcpayserver-zcash-plugin)  
<img src="https://docs.btcpayserver.org/img/btcpay-logo.svg" alt="BTCPay Server Logo" width="200"/>

---

## ZGo
- **Тип поддержки**: Transparent and Shielded   
- **Описание**: ZGo — это электронная платёжная платформа, где платежи поступают напрямую от вашего клиента к вам, без участия третьих сторон.  
- **URL**: [ZGo](https://zgo.cash/)  
<img src="https://i.ibb.co/FLhpwnz5/zgo-prp2.png" alt="ZGo Logo" width="200"/>

---

## NOWPayments
- **Тип поддержки**: Transparent   
- **Описание**: NOWPayments — это криптоплатёжный шлюз, который позволяет продавцам легко принимать платежи и пожертвования в Zcash.  
- **URL**: [NOWPayments](https://nowpayments.io/supported-coins/zcash-payments)  
<img src="https://nowpayments.io/images/logo/logo.svg" alt="NOWPayments Logo" width="200"/>

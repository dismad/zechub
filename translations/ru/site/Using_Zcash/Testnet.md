# Zcash Testnet

## Что такое Zcash Testnet?

**Zcash Testnet** — это параллельный блокчейн к реальной основной сети Zcash (Mainnet), который воспроизводит тот же самый протокол, правила и логику транзакций, но с двумя ключевыми отличиями:

1. **Монеты не имеют реальной денежной ценности** — они называются **TAZ**, а не ZEC, и используются только для тестирования.  
2. **Обновления сети, инструменты и программное обеспечение сначала тестируются здесь** перед развёртыванием в реальном блокчейне Zcash.  

Иными словами, Testnet — это своего рода **песочница или экспериментальная среда**, где разработчики, аудиторы и создатели могут проверять идеи, не рискуя реальными деньгами.


## Зачем существует Testnet?

Testnet крайне важен для разработки блокчейнов, потому что **реальные блокчейны, такие как Zcash, неизменяемы** — как только транзакции подтверждены в основной сети, их нельзя отменить. Testnet предоставляет **безопасную копию** для экспериментов, тестирования и отладки функций перед их развёртыванием в Mainnet.

### Варианты использования Testnet

#### 1. Разработка и интеграция программного обеспечения

Разработчики, создающие кошельки, биржи, ПО для майнинга или инструменты приватности, могут безопасно тестировать их в Testnet. Возможности включают:

- Отправку и получение транзакций  
- Майнинг новых блоков с монетами TAZ нулевой стоимости  
- Создание пользовательских интерфейсов и API  
- Тестирование функций приватности транзакций (прозрачные и shielded)  

**Пример:**  
Такие инструменты, как [`zcash_tx_tool`](https://github.com/QED-it/zcash_tx_tool), используют Testnet для создания транзакций и тестирования функциональности shielded-активов Zcash.  

**Сценарий из реальной практики:**  
Разработчик кошелька может подключить программное обеспечение к RPC-узлу Testnet и смоделировать полный жизненный цикл — создание адресов, отправку shielded-транзакций и проверку балансов — прежде чем запускать решение в Mainnet.

#### 2. Тестирование обновлений сети

Zcash периодически обновляет свой базовый протокол (например, Nu5, Nu6). В Testnet новые обновления активируются **раньше, чем в Mainnet**, что позволяет разработчикам и сообществу выявлять и исправлять ошибки.

**Пример:**  
Новое правило консенсуса или новый тип транзакции сначала внедряется в Testnet. После успешного тестирования оно активируется в Mainnet на заранее определённой высоте блока.

#### 3. Тестирование реализаций нод

Zcash поддерживает несколько реализаций программного обеспечения нод — `zcashd` и **Zebra** (нода на Rust, поддерживаемая Zcash Foundation). Testnet позволяет тестировать ноды в реальных условиях без финансового риска.  

Разработчики нод могут:

- Проверять распространение блоков  
- Тестировать RPC-интерфейсы  
- Наблюдать за поведением нод под нагрузкой  
- Тестировать взаимодействие с майнинговым ПО  

#### 4. Обучение и образование

Новички могут изучать возможности Zcash, такие как майнинг, создание shielded-транзакций и использование Unified Address.  
Учебные материалы и документация сообщества предоставляют доступ к **кранам Testnet, обозревателям и руководствам**.


## Реальные сценарии использования Testnet

### 1. Тестирование разработчиками (кошелёк / приложение)

- Подключение к Zcash Testnet  
- Запрос TAZ из крана  
- Отправка shielded-транзакций  
- Проверка приватности и стабильности интерфейса  

Даже если произойдут ошибки, реальные ZEC не будут потеряны.

### 2. Тестирование интеграции с биржей

- Запуск ноды Testnet  
- Использование JSON-RPC-эндпоинтов Zebrad для обработки транзакций  
- Тестирование автоматизированной логики депозитов и вывода средств  

Это обеспечивает безопасность production-кода и предотвращает финансовые потери.

### 3. Испытания настройки майнинга

- Использование шаблонов для майнинга  
- Тестирование валидации блоков  
- Наблюдение за наградами за майнинг (только TAZ)  
- Настройка производительности майнинга  

Это помогает избежать простоев или потери дохода при переходе в Mainnet.

### 4. Академические исследования / исследования протокола

Исследователи могут тестировать инновации, такие как **stateless verification**, **оптимизация доказательств с нулевым разглашением** и другие протокольные эксперименты с использованием Testnet.  
Продвинутые пользователи также могут запускать **собственные Testnet или окружения regtest** для специализированных экспериментов.


## Ключевые различия между Mainnet и Testnet

| Feature               | Mainnet           | Testnet                  |
|-----------------------|-----------------|--------------------------|
| Ценность монет        | Реальные ZEC     | TAZ (без денежной ценности) |
| Риск                  | Финансовый риск  | Безопасно для тестирования |
| Обновления протокола  | Production       | Ранняя активация         |
| Награды за майнинг    | Реальная эмиссия | Только тестовая награда  |
| Назначение сети       | Реальные транзакции| Тестирование и разработка |

## Распространённые заблуждения

- **Монеты Testnet чего-то стоят** -> Неверно, TAZ не имеют ценности.  
- **Потеря монет Testnet имеет значение** -> Неверно, реальная ценность не теряется.  
- **Testnet и Mainnet идентичны** -> Неверно, Testnet часто сбрасывается и не имеет такой экономической защищённости, как Mainnet.

## Что такое TAZ?

**TAZ** — это версия монет Zcash для Testnet:  

- Это не реальные деньги; их нельзя обменять на ZEC или фиат  
- Используются для тестирования, разработки и обучения  
- Следуют всем правилам Zcash: их можно отправлять, майнить и использовать на shielded-адресах  

**Пример:**  
Разработчик может отправить 100 TAZ с одного адреса Testnet на другой, чтобы протестировать функцию кошелька, не рискуя реальными ZEC.  

Думайте о TAZ как об **«игровых деньгах» для Zcash Testnet**.


## Что такое краны?

**Кран** — это сервис, который бесплатно выдаёт монеты TAZ для тестирования:

- Обычно это веб-сайты или API  
- Пользователь указывает адрес Testnet, и кран отправляет небольшое количество TAZ  
- Это избавляет от необходимости майнить TAZ вручную  

**Пример:**  
1. Посетите кран Testnet (например, [testnet.zecfaucet.com](https://testnet.zecfaucet.com) | [fauzec.com](https://fauzec.com/))  
2. Введите ваш адрес Testnet  
3. Запросите TAZ  
4. Мгновенно получите TAZ и начните тестирование  

**Почему это важно:**  
- Безопасное тестирование без риска для ZEC  
- Доступность для новичков и разработчиков  
- Быстрое прототипирование для кошельков, бирж и приложений



## Кошельки Zkool и Zingo!

### Zkool

- Мультиаккаунтный кошелёк для продвинутых пользователей Zcash  
- Поддерживает seed-фразы, viewing keys, прозрачные и shielded-адреса  
- Может подключаться к Mainnet, Testnet или Regtest через полные ноды или lightwallet-серверы

### Zingo!

- Мобильный кошелёк, ориентированный на приватность и простоту  
- Поддерживает shielded- и unified-адреса  
- Обновлён для поддержки протоколов Testnet (включая NU6 Testnet)

## После включения Testnet

- Транзакции работают как в Mainnet, но с **TAZ нулевой стоимости**  
- Можно тестировать shielded-транзакции, несколько адресов и функции приватности  
- Разработчики могут отлаживать и тестировать функции, не рискуя реальными ZEC


## Краткое резюме

- **Zcash Testnet** — это безопасная среда-песочница для разработки, тестирования и экспериментов  
- Сценарии использования: тестирование разработчиками, тестирование нод, интеграция с биржами, исследования и обучение  
- Вместо ZEC используются **монеты TAZ**, которые не имеют реальной ценности  
- Testnet необходим перед развёртыванием функций в Mainnet

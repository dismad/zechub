## Zcash.me

![Логотип Zcash.me](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### Обзор

**Zcash.me** — это каталог профилей и платформа для поиска, которая позволяет любому человеку легко находить пользователей Zcash и отправлять им ZEC. Она служит «Linktree для Zcash» — простым способом поделиться своим shielded-адресом и присутствием в социальных сетях с помощью одной ссылки.

| Пункт             | Подробности |
|-------------------|---------|
| **Сайт**          | [https://zcash.me/](https://zcash.me/) |
| **Twitter**       | [@zcashme](https://x.com/zcashme) |
| **Формат профиля**| `zcash.me/[username]` |
| **Статус**        | Активен (функции находятся в разработке) |

### Ключевые возможности

#### Доступно сейчас
- **Пользовательские страницы профиля** — персонализированная страница с именем пользователя, биографией, аватаром и местоположением
- **Отображение shielded-адреса** — кнопка копирования вашего Unified Address в один клик
- **Интеграция социальных ссылок** — X, LinkedIn, Telegram и пользовательские веб-сайты
- **Система верификации** — значок подтверждения для доверия сообщества
- **Таблица лидеров по рефералам** — отслеживание лучших рефереров и лидеров сообщества
- **Статус активности** — показывает время последней активности

#### Скоро появится
| Возможность        | Описание |
|--------------------|-------------|
| Карта местоположений | Найти пользователей Zcash поблизости |
| Статистика платформы| Просмотр метрик роста |
| Опросы сообщества   | Голосование по темам экосистемы |
| Новости и объявления | Официальные обновления |
| Форум              | Обсуждения сообщества |
| Мобильное приложение | Полноценный мобильный интерфейс |

### Как начать
1. Перейдите на [https://zcash.me/](https://zcash.me/) и нажмите **Join**
2. Выберите уникальное имя пользователя, добавьте биографию и фото профиля
3. Вставьте свой Zcash Unified Address (`u1...`)
4. Подключите свои социальные аккаунты
5. Поделитесь своей ссылкой: `zcash.me/[yourusername]`

## TipZ

![Логотип TipZ](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### Обзор

**TipZ** — это **приватный протокол для чаевых**, специально созданный для создателей контента и cypherpunks. Поклонники отправляют чаевые в любой крупной криптовалюте; они автоматически конвертируются и доставляются в виде **shielded ZEC**.

| Пункт             | Подробности |
|-------------------|---------|
| **Сайт**          | [https://tipz.cash/](https://tipz.cash/) |
| **Twitter**       | [@tipz_cash](https://x.com/tipz_cash) |
| **Формат профиля**| `tipz.cash/@[username]` |
| **Статус**        | Активен |

> *«tipz.cash — это приватный протокол для чаевых для создателей контента и cypherpunks.»*

### Почему приватность важна для создателей контента

#### Финансовая свобода
- 0% комиссии платформы
- Некостодиальный подход и самостоятельное хранение средств
- Без посредников

#### Дизайн с приоритетом приватности
- Каждые чаевые поступают как shielded ZEC
- Приватный баланс кошелька
- Зашифрованные сообщения вместе с чаевыми
- Полная опциональная анонимность создателя контента

#### Кроссчейн-платежи
| Валюта | Сеть |
|----------|---------|
| ETH      | Ethereum |
| USDC     | Несколько сетей |
| SOL      | Solana |
| ZEC      | Напрямую через Zcash |

#### Защита создателей контента
- Нет риска деплатформинга
- Устойчивость к цензуре
- KYC не требуется

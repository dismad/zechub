## Zcash.me

![شعار Zcash.me](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### نظرة عامة

**Zcash.me** هو دليل ملفات شخصية ومنصة اكتشاف تجعل من السهل على أي شخص العثور على مستخدمي Zcash وإرسال ZEC إليهم. وهو بمثابة "Linktree لـ Zcash" — طريقة بسيطة لمشاركة عنوانك المحمي وحضورك على وسائل التواصل عبر رابط واحد.

| العنصر              | التفاصيل |
|-------------------|---------|
| **الموقع الإلكتروني**       | [https://zcash.me/](https://zcash.me/) |
| **تويتر**       | [@zcashme](https://x.com/zcashme) |
| **صيغة الملف الشخصي**| `zcash.me/[username]` |
| **الحالة**        | متاح الآن (مع ميزات قيد التطوير) |

### الميزات الرئيسية

#### المتاح حاليًا
- **صفحات ملفات شخصية مخصصة** - صفحة مخصصة تتضمن اسم المستخدم، والنبذة، والصورة الرمزية، والموقع
- **عرض العنوان المحمي** - زر نسخ بنقرة واحدة لـ Unified Address الخاص بك
- **دمج الروابط الاجتماعية** - X وLinkedIn وTelegram ومواقع الويب المخصصة
- **نظام التحقق** - شارة موثقة لتعزيز ثقة المجتمع
- **لوحة صدارة الإحالات** - تتبع أبرز المُحيلين وقادة المجتمع
- **حالة النشاط** - تعرض وقت آخر نشاط

#### قريبًا
| الميزة            | الوصف |
|--------------------|-------------|
| خريطة المواقع       | العثور على مستخدمي Zcash القريبين |
| إحصاءات المنصة| عرض مؤشرات النمو |
| استطلاعات المجتمع    | التصويت على موضوعات النظام البيئي |
| الأخبار والإعلانات | تحديثات رسمية |
| المنتدى              | نقاشات المجتمع |
| تطبيق الهاتف المحمول         | تجربة متكاملة على الهاتف المحمول |

### البدء
1. زر [https://zcash.me/](https://zcash.me/) وانقر على **Join**
2. اختر اسم مستخدم فريدًا، وأضف نبذة وصورة للملف الشخصي
3. الصق Unified Address الخاص بك على Zcash (`u1...`)
4. اربط حساباتك الاجتماعية
5. شارك رابطك: `zcash.me/[yourusername]`

## TipZ

![شعار TipZ](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### نظرة عامة

**TipZ** هو **بروتوكول إكراميات خاص** صُمم خصيصًا لمنشئي المحتوى والسيبرفانكس. يرسل المعجبون الإكراميات بأي عملة مشفرة رئيسية؛ ثم تُحوَّل تلقائيًا وتُسلَّم على شكل **ZEC محمي**.

| العنصر              | التفاصيل |
|-------------------|---------|
| **الموقع الإلكتروني**       | [https://tipz.cash/](https://tipz.cash/) |
| **تويتر**       | [@tipz_cash](https://x.com/tipz_cash) |
| **صيغة الملف الشخصي**| `tipz.cash/@[username]` |
| **الحالة**        | متاح الآن |

> *"tipz.cash هو بروتوكول إكراميات خاص لمنشئي المحتوى والسيبرفانكس."*

### لماذا تهم الخصوصية لمنشئي المحتوى

#### الحرية المالية
- 0% رسوم منصة
- غير وصائي وحفظ ذاتي
- بلا وسطاء

#### تصميم يضع الخصوصية أولًا
- كل إكرامية تصل على شكل ZEC محمي
- رصيد محفظة خاص
- مراسلة مشفرة مع الإكراميات
- خيار إخفاء هوية منشئ المحتوى بالكامل

#### مدفوعات عبر سلاسل متعددة
| العملة | الشبكة |
|----------|---------|
| ETH      | Ethereum |
| USDC     | سلاسل متعددة |
| SOL      | Solana |
| ZEC      | Zcash مباشر |

#### حماية منشئي المحتوى
- لا خطر من الإزالة من المنصة
- مقاوم للرقابة
- لا يتطلب KYC

# Blockchain Nhwehwɛmufo

## Dɛn ne Explorer?

Blockchain explorers yɛ nnwinnade a ɛma obi tumi yɛ nea asi dedaw wɔ blockchain so ho mfonini wɔ n’adwenem ara kwa. Wɔfa ma inputs: transaction id's, addresses, anaa block hashes, na wɔde aniwa kyerɛ nea ɛkɔɔ so.

## Zcash Block Explorers a wɔahyehyɛ wɔn:

- [Zcash Block Nhwehwɛmufoɔ](https://mainnet.zcashexplorer.app/)

- [Akongua a wɔde si dan mu](https://blockchair.com)

- [3xpl](https://3xpl.com/zcash)

- [Bitquery, asɛm a wɔka kyerɛ](https://explorer.bitquery.io/zcash)

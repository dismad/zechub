# Ndị na-enyocha Blockchain

## Gịnị bụ Explorer?

Blockchain explorers bụ ngwa ọrụ na-enye mmadụ ohere iji anya nke uche hụ ihe emeworị na blockchain. Ha na-ewere maka ntinye: azụmahịa id's, adreesị, ma ọ bụ ngọngọ hashes, ma jiri anya pụta ihe mere.

## Ndepụta nke Zcash Block Explorers:

- [Zcash Block Explorer](https://mainnet.zcashexplorer.app/)

- [Otu oche](https://blockchair.com)

- [3xpl](https://3xpl.com/zcash)

- [Bitquery](https://explorer.bitquery.io/zcash)

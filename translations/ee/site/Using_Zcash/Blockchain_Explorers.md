# Blockchain ƒe Nukulawo

## Nukae nye Explorer?

Blockchain explorers nye dɔwɔnu siwo nana ame kpɔa nusiwo dzɔ xoxo le blockchain dzi le susu me ko. Woxɔa na nusiwo wotsɔ de eme: asitsatsa ƒe id's, adrɛswo, alo block hashes, eye woɖea nusi dzɔ la fiana le nukpɔkpɔ me.

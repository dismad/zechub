# Nukae Nye Zcash TEX Adrɛswo?

Zcash TEX adrɛswo tsi tre ɖi na adrɛs si woxɔna ƒe ƒomevi tɔxɛ aɖe. "Transparent Exchange" adrɛs ƒe kpukpui, enye **Etɔxɛ**, Unified-ƒomevi (bech32m) ƒe nuŋɔŋlɔ si nye p2pkh Transparent adrɛs ɖeka. 

Eƒe taɖodzinu ɖeka koe nye be wòana gakotoku si sɔ la nanya be wòawɔ Transparent-Only (T -> T) ƒe asitsatsa. 

Susu si le emee nye esi: Ne wokpɔ TEX Adrɛs aɖe la, gakotoku si sɔ la ɖea eƒe nyawo gɔme be wòaxɔ Transparent receiver si le eme. Emegbe gakotokua ɖoa ga si hiã na tx la tso Shielded pool la me yia adrɛs si le vovo, si dzi zãla kpɔna, si nɔa anyi ɣeyiɣi kpui aɖe ko si nye Transparent (Z -> T). Emegbe eɖoa ga mawo ɖe TEX adrɛs (T -> T) ƒe Transparent receiver si woɖe tso eme. 

Wogblɔ mɔ̃ɖaŋununya ƒe aɖaŋuɖoɖo na TEX adrɛswo le Zcash [ZIP 320](https://zips.z.cash/zip-0320), si ɖea adrɛs ƒomevi aɖe si woatsɔ axɔ ga tso Adrɛs siwo me kɔ gbɔ ko.

![TEX](https://i.ibb.co/8m7HPqV/ZashiTex.png)


Togbɔ be woxɔa TEX adrɛs siwo menye le afisiafi o hã la, woate ŋu abia tso Zcash zãlawo si be woazã wo mlɔeba.

## Ɣekaɣie Mehiã TEX Adrɛs

### **Èhiã** TEX adrɛs ne èle ga ɖom ɖe Transparent adrɛs to gakotoku si mewɔa dɔ le eɖoɖo ɖe Transparent adrɛs tẽ o zazã me. 
Ðeko gakotoku aɖewo meɖea mɔ be woaɖoe ɖe Transparent adrɛs tẽ o eye **ate ŋu adzɔ be amesi xɔe la matsɔ TEX si sɔ kplii o**. Eyata, **Trɔtrɔ** tso Transparent dzi yi TEX adrɛs dzi ate ŋu abia ɣeaɖewoɣi. Woateŋu awɔ esia kple asi to nufiame ƒe dɔwɔwɔ si wogblɔ le zip-320 me la wɔwɔ me. Woateŋu akpɔ **Transparent-to-TEX-Converter** ƒe kpɔɖeŋu si woxɔ [AFI SIA](https://690e9524c66a3ecac5d54eff--jade-brioche-873777.netlify.app/).

### Èhiã TEX adrɛs ne èle ga ɖom ɖe centralized exchange si **BIA be ga mawo natso Transparent dzɔtsoƒe**. 
Fifia la, [Binance](https://www.binance.com/) nye Centralized Exchange ɖeka kolia si zãa TEX adrɛswo (eye woawoe nye susu vevitɔ si ta wowɔ TEX ɖo). 
TEX adrɛswo naa gakotoku si sɔ la nya be ele be ga siwo katã woɖo ɖe adrɛs ma dzi la nadze ƒã eye woaɖe asixɔxɔ ɖesiaɖe si wokpɔ ta na la ɖa be woagaɖo wo ɖe adrɛs si woyɔ la dzi o.
Ne asitɔtrɔ abe Binance ene gbe asixɔxɔ si woɖo ɖa la, mɔnu si hiã le esi be wòatrɔ asixɔxɔ ma atrɔ ayi adrɛs si dzi wòtso la dzi. Ekpena ɖe dɔwɔƒewo abe Binance ene hã ŋu be woawɔ ɖe se kple ɖoɖo siwo dziɖuɖuwo alo dziɖuɖumegã bubuwo de dzi.

# Zcash-Zahlungsabwickler

## CoinPayments
- **Unterstützungstyp**: Transparent   
- **Beschreibung**: Ein globales Kryptowährungs-Zahlungsgateway, das über 100 Kryptowährungen unterstützt, darunter Zcash. Bietet eine einfache Integration für Onlineshops.  
- **URL**: [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="CoinPayments-Logo" width="200"/>

---

## CipherPay
- **Unterstützungstyp**: Shielded   
- **Beschreibung**: Akzeptieren Sie Zcash in wenigen Minuten, Non-custodial, keine Käuferdaten, kein Mittelsmann.  
- **URL**: [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="CipherPay-Logo" width="200"/>

---

## Flexa
- **Unterstützungstyp**: Transparent 
- **Beschreibung**: Flexa ist ein Kryptowährungs-Zahlungsabwickler, der es Händlern ermöglicht, digitale Vermögenswerte, einschließlich Zcash (ZEC), für alltägliche Transaktionen zu akzeptieren.  
- **URL**: [Flexa](https://flexa.co/)  
<img src="https://framerusercontent.com/images/7D7WwOyaY4PFVW3ZiBzGggx13Xo.svg" alt="Flexa-Logo" width="200"/>

---

## BTCPay Server
- **Unterstützungstyp**: Shielded   
- **Beschreibung**: BTCPay Server ist ein Open-Source-Kryptowährungs-Zahlungsabwickler zum Selbsthosten.

## ZGo
- **Unterstützungstyp**: Transparent und Shielded   
- **Beschreibung**: ZGo ist eine elektronische Zahlungsplattform, die direkt von Ihrem Kunden zu Ihnen geht, ohne dass Dritte beteiligt sind.  
- **URL**: [ZGo](https://zgo.cash/)

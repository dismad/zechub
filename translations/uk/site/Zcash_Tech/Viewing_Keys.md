# Viewing Key

Екрановані адреси дають змогу користувачам здійснювати транзакції, розкриваючи якомога менше інформації в блокчейні Zcash. Що відбувається, коли вам потрібно розкрити конфіденційну інформацію, пов’язану з екранованою транзакцією Zcash, конкретній стороні? Кожна екранована адреса містить viewing key. Viewing key були представлені в [ZIP 310](https://zips.z.cash/zip-0310) і додані до протоколу в оновленні мережі Sapling. Viewing key є важливою частиною Zcash, оскільки вони дають користувачам змогу вибірково розкривати інформацію про транзакції.

### Навіщо використовувати viewing key?

Чому користувач узагалі хотів би це робити? Ось що про це йдеться в блозі Electric Coin Co....

*- Біржа хоче визначати, коли клієнт вносить ZEC на екрановану адресу, водночас зберігаючи ключі **spend authority** на захищеному обладнанні. Біржа може згенерувати incoming viewing key і завантажити його на підключений до Інтернету вузол **detection**, тоді як spending key залишатиметься в більш захищеній системі.*

*- Кастодіан має забезпечити аудиторам видимість своїх активів Zcash. Кастодіан може згенерувати full viewing key для кожної зі своїх екранованих адрес і надати цей ключ аудитору. Аудитор зможе перевірити баланс цих адрес і переглянути попередню активність транзакцій до цих адрес і з них.* 

*- Біржі може знадобитися провести належну перевірку клієнта, який здійснює депозити з екранованої адреси. Біржа може запросити viewing key клієнта для його екранованої адреси та використати його для перегляду екранованої транзакційної активності клієнта в межах цих розширених процедур належної перевірки.*

### Як знайти свій viewing key

#### zcashd

* Перелічіть усі відомі адреси за допомогою *./zcash-cli listaddresses*

* Потім виконайте таку команду для UA або екранованих адрес Sapling

  ```bash
  ./zcash-cli z_exportviewingkey "<UA or Z address>"
  ```

#### Ywallet

* У верхньому правому куті виберіть "Backup", пройдіть автентифікацію на телефоні, а потім просто скопіюйте свій viewing key, який буде показано.

### Як використовувати свій viewing key

#### zcashd

* Використовуйте таку команду з будь-яким vkey або ukey: 

```bash
./zcash-cli z_importviewingkey "vkey/ukey" whenkeyisnew 30000
```

#### ywallet

* У верхньому правому куті виберіть "Account", натисніть "+" у нижньому правому куті, щоб додати та імпортувати свій viewing key і додати свій обліковий запис лише для читання ('read-only').

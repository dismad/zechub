# Нотатки

#### Надсилання зашифрованих нотаток

Під час надсилання транзакції Z2Z (від shielded до shielded) ви можете додати до транзакції нотатку (повідомлення). Цю нотатку можна використовувати для різних цілей.

#### Підписування транзакцій

Нотатки насамперед використовуються для підписування платежів. Оскільки shielded-транзакції шифрують ваші дані, ви не можете побачити, хто надіслав вам ZEC і для чого могли бути призначені ці ZEC. Користувачі можуть використовувати поле нотатки, щоб підписати його своїм ім’ям або псевдонімом і дати іншій стороні знати, від кого була транзакція. Вони також можуть описати, для чого була ця транзакція.

#### Надсилання повідомлення

Ще один варіант використання зашифрованої нотатки — надсилання повідомлення комусь, хто має z-addr. Ці повідомлення можуть бути про що завгодно: чи то [нагадування для друга](https://twitter.com/iansagstette/status/1542142468505870336), чи [чутливе повідомлення, яке має залишатися настільки приватним, наскільки це можливо](https://twitter.com/InsideZcash/status/1545800146352578560).

#### Любовні записки в блокчейні

Була людина, яка надіслала своєму партнеру любовну записку в одному з перших блоків блокчейна Zcash. Хтось виявив, що їхній партнер надіслав їм файл через нотатку Zcash. Цей файл був квитком на особливу подію за кордоном, яку вона та її коханий на відстані обговорювали й планували відвідати разом. Нотатка була любовним посланням.

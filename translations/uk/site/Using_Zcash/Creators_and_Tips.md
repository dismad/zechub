## Zcash.me

![Логотип Zcash.me](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### Огляд

**Zcash.me** — це каталог профілів і платформа для пошуку, яка дозволяє будь-кому легко знаходити користувачів Zcash і надсилати їм ZEC. Вона слугує "Linktree для Zcash" — простим способом ділитися своєю shielded-адресою та присутністю в соцмережах за допомогою одного посилання.

| Елемент              | Деталі |
|----------------------|--------|
| **Вебсайт**          | [https://zcash.me/](https://zcash.me/) |
| **Twitter**          | [@zcashme](https://x.com/zcashme) |
| **Формат профілю**   | `zcash.me/[username]` |
| **Статус**           | Працює (з функціями в розробці) |

### Ключові можливості

#### Наразі доступно
- **Користувацькі сторінки профілю** — персоналізована сторінка з іменем користувача, біографією, аватаром і локацією
- **Відображення shielded-адреси** — кнопка копіювання вашої unified address в один клік
- **Інтеграція соціальних посилань** — X, LinkedIn, Telegram і власні вебсайти
- **Система верифікації** — позначка верифікації для довіри спільноти
- **Реферальна таблиця лідерів** — відстеження найкращих рефералів і лідерів спільноти
- **Статус активності** — показує час останньої активності

#### Незабаром
| Функція              | Опис |
|----------------------|------|
| Карта локацій        | Пошук користувачів Zcash поблизу |
| Статистика платформи | Перегляд показників зростання |
| Опитування спільноти | Голосування за теми екосистеми |
| Новини та оголошення | Офіційні оновлення |
| Форум                | Обговорення спільноти |
| Мобільний застосунок | Повноцінний мобільний досвід |

### Початок роботи
1. Перейдіть на [https://zcash.me/](https://zcash.me/) і натисніть **Join**
2. Оберіть унікальне ім’я користувача, додайте біографію та фото профілю
3. Вставте свою unified address Zcash (`u1...`)
4. Під’єднайте свої соціальні акаунти
5. Поділіться своїм посиланням: `zcash.me/[yourusername]`

## TipZ

![Логотип TipZ](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### Огляд

**TipZ** — це **приватний протокол для чайових**, спеціально створений для творців і cypherpunks. Прихильники надсилають чайові в будь-якій великій криптовалюті; вони автоматично конвертуються та доставляються як **shielded ZEC**.

| Елемент              | Деталі |
|----------------------|--------|
| **Вебсайт**          | [https://tipz.cash/](https://tipz.cash/) |
| **Twitter**          | [@tipz_cash](https://x.com/tipz_cash) |
| **Формат профілю**   | `tipz.cash/@[username]` |
| **Статус**           | Працює |

> *"tipz.cash is a private tipping protocol for creators and cypherpunks."*

### Чому приватність важлива для творців

#### Фінансова свобода
- 0% комісій платформи
- Non-custodial і self-custody
- Без посередників

#### Дизайн із пріоритетом приватності
- Кожен чайовий платіж надходить як shielded ZEC
- Приватний баланс гаманця
- Зашифровані повідомлення разом із чайовими
- Повна опція анонімності для творця

#### Кросчейн-платежі
| Валюта | Мережа |
|--------|--------|
| ETH    | Ethereum |
| USDC   | Кілька мереж |
| SOL    | Solana |
| ZEC    | Direct Zcash |

#### Захист творців
- Відсутність ризику деплатформінгу
- Стійкість до цензури
- KYC не потрібен

### Додаткові ресурси

```
TipZ Payment Flow

Fan -> tipz.cash Protocol -> Creator's Wallet (Shielded ZEC)

Fan selects amount + message -> Protocol auto-converts -> Creator receives private shielded ZEC
```

# Платіжні процесори Zcash

## CoinPayments
- **Тип підтримки**: Transparent   
- **Опис**: Глобальний платіжний шлюз для криптовалют, який підтримує понад 100 криптовалют, включно з Zcash. Пропонує просту інтеграцію для онлайн-магазинів.  
- **URL**: [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="CoinPayments Logo" width="200"/>

---

## CipherPay
- **Тип підтримки**: Shielded   
- **Опис**: Приймайте Zcash за лічені хвилини, без кастодіального зберігання, без даних покупця, без посередників.  
- **URL**: [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="CipherPay Logo" width="200"/>

---

## Flexa
- **Тип підтримки**: Transparent 
- **Опис**: Flexa — це процесор криптовалютних платежів, який дає змогу продавцям приймати цифрові активи, зокрема Zcash (ZEC), для щоденних транзакцій.  
- **URL**: [Flexa](https://flexa.co/)  
<img src="https://framerusercontent.com/images/7D7WwOyaY4PFVW3ZiBzGggx13Xo.svg" alt="Flexa Logo" width="200"/>

---

## BTCPay Server
- **Тип підтримки**: Shielded   
- **Опис**: BTCPay Server — це платіжний процесор криптовалют із відкритим кодом, який розміщується самостійно.  
- **URL**: [BTCPay Server](https://github.com/btcpay-zcash/btcpayserver-zcash-plugin)  
<img src="https://docs.btcpayserver.org/img/btcpay-logo.svg" alt="BTCPay Server Logo" width="200"/>

---

## ZGo
- **Тип підтримки**: Transparent and Shielded   
- **Опис**: ZGo — це електронна платіжна платформа, де оплата надходить напряму від вашого клієнта до вас, без участі третіх сторін.  
- **URL**: [ZGo](https://zgo.cash/)  
<img src="https://i.ibb.co/FLhpwnz5/zgo-prp2.png" alt="ZGo Logo" width="200"/>

---

## NOWPayments
- **Тип підтримки**: Transparent   
- **Опис**: NOWPayments — це криптоплатіжний шлюз, який дає змогу продавцям легко приймати платежі та пожертви в Zcash.  
- **URL**: [NOWPayments](https://nowpayments.io/supported-coins/zcash-payments)  
<img src="https://nowpayments.io/images/logo/logo.svg" alt="NOWPayments Logo" width="200"/>

# <img src="https://i.ibb.co/bjYy15Yb/ZCG-Circle-Logo-10.png" alt="Alt Text" width="50"/>  Гранти спільноти Zcash

## Місія

Програма грантів спільноти Zcash фінансує незалежні команди, які входять до екосистеми Zcash, для виконання масштабної довгострокової розробки (або іншої роботи) на благо всієї екосистеми Zcash. 

## Люди та команди

Грантами спільноти Zcash керує комітет, який наразі складається з п’яти осіб. Ця спільнота розглядає та схвалює грантові заявки від сторонніх команд, які хочуть розвивати протокол Zcash. Більше інформації про [команду](https://zcashcommunitygrants.org/committee/)

Нещодавні проєкти, схвалені ZCG, включають Zcash Shielded Assets (ZSA), який очолює команда QEDIT. Впровадження DeFi для ZEC у Zcash за допомогою нового платіжного протоколу, що додає додаткові можливості до основної мережі Zcash. 

Короткометражний документальний фільм Zcash Media за участю відомих прихильників Zcash, таких як Edward Snowden, Zooko і Deirdre Connolly, з метою створення актуальних, захопливих і освітніх відео для широкого поширення. 

Інтеграція апаратних гаманців Ledger для повної підтримки захищених транзакцій Sapling, що дозволяє зберігати Zcash приватно та безпечно. Це було реалізовано командою Zondax. 

Проста система захищених платежів в один клік і система point-of-sale для фізичних магазинів у форматі SDK, над якою працює ZGo. 

Програма Global Ambassador допомагає Zcash отримати ширше міжнародне представництво та бути представленим на офлайн-заходах спільноти по всьому світу. 

Ще кілька проєктів було розглянуто Комітетом грантів Zcash, дивіться [повний оновлений список](https://docs.google.com/spreadsheets/d/1FQ28rDCyRW0TiNxrm3rgD8ai2KGUsXAjPieQmI1kKKg/edit?gid=135980745#gid=135980745) поданих проєктів.

## Організаційна структура

Гранти спільноти Zcash є незалежною організацією від Electric Coin Company та Zcash Foundation. 

## Фінансування

Electric Coin Company отримує 8% від [фонду розвитку Zcash](https://zips.z.cash/zip-1014). Ці кошти використовуються для підтримки сторонніх команд розробників, а також для внутрішніх операцій.

### Крок 1: Підготуйте свою пропозицію
Створіть чітку пропозицію, яка включає:
- Назву проєкту та дані заявника
- Опис проблеми та запропоноване рішення
- Результати, графік і ключові етапи
- Бюджет і запит на фінансування
- Відповідний досвід і ризики

---

### Крок 2: Подайте заявку на GitHub
1. Перейдіть до **репозиторію GitHub Zcash Community Grants**
2. Відкрийте вкладку **Issues** і натисніть **New Issue**
3. Виберіть відповідний **шаблон грантової заявки**
4. Заповніть усі обов’язкові поля та надішліть заявку

Issue на GitHub є **офіційним записом заявки**.

---

### Крок 3: Створіть допис на форумі спільноти
1. Перейдіть на **форум спільноти Zcash**
2. Створіть нову тему в категорії **Zcash Community Grants**
3. Використайте ту саму назву, що й в issue на GitHub
4. Додайте короткий підсумок і вставте **посилання на issue на GitHub**

---

### Крок 4: Долучайтеся до розгляду
- Слідкуйте за issue на GitHub і темою на форумі
- Відповідайте на запитання та відгуки від комітету ZCG і спільноти

---

### Крок 5: Результат
- Схвалені пропозиції переходять до фінансування та звітування
- Несхвалені пропозиції **МОЖУТЬ** бути переглянуті та подані повторно

---

**Примітка:** GitHub є основним джерелом запису; форум призначений для публічного обговорення.

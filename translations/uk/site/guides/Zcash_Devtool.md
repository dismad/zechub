# Zcash Devtool

[Що таке Zcash-Devtool?](https://github.com/zcash/zcash-devtool?tab=readme-ov-file) 

Zcash Devtool — це платформа для роботи над Zcash. Її створили розробники для розробників, для тестування та розробки нової функціональності Zcash; і її не слід вважати готовою до використання у продакшені. API командного рядка, який надає цей інструмент, може змінюватися і буде змінюватися в будь-який час і без попередження. НЕ довіряйте керуванню значними коштами вбудованому гаманцю zcash-devtool.

Щоб ознайомитися з покроковим посібником із початку роботи з цими інструментами, дивіться [цей walkthrough](https://github.com/zcash/zcash-devtool/blob/main/doc/walkthrough.md). У ньому задокументовано повний покроковий процес налаштування та використання інструментарію zcash devtool. Він призначений слугувати посібником із налаштування, а також пояснює, як додавати до інструмента власну функціональність.

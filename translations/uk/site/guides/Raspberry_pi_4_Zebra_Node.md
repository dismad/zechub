# Посібник із запуску Zebra на Raspberry Pi 4

<img src="https://i.ibb.co/V3rjKwv/image-2023-11-28-172907488.png" alt="raspberry pi" width="300" height="300"/>

Запуск програмного забезпечення вузла Zebra на Raspberry Pi 4 дає змогу брати участь у мережі Zcash як незалежний вузол, сумісний із консенсусом. Цей посібник проведе вас через кроки налаштування та запуску Zebra на вашому Raspberry Pi 4.

## Передумови

1. Raspberry Pi 4 (рекомендовано 2 ГБ RAM або більше).

2. Карта MicroSD (рекомендовано 16 ГБ або більше) із встановленою Raspberry Pi OS (Raspbian).

3. Стабільне інтернет-з’єднання.

4. Клавіатура, миша та монітор (для початкового налаштування).

5. SSH-клієнт (необов’язково, для віддаленого доступу).

## Встановлення

1. __Оновіть вашу систему__
   Відкрийте термінал або підключіться до Raspberry Pi через SSH і переконайтеся, що ваша система актуальна, виконавши:

   __sudo apt update__

   __sudo apt upgrade__

2. __Встановіть залежності__
   Вам потрібно буде встановити деякі необхідні залежності для збирання та запуску Zebra:

   __sudo apt install build-essential cmake git clang libssl-dev pkg-config__

3. __Клонуйте репозиторій Zebra__
   Відкрийте термінал і клонуйте репозиторій Zebra на ваш Raspberry Pi:

   __git clone https://github.com/ZcashFoundation/zebra.git__

   __cd zebra__

4. __Зберіть Zebra__
   Щоб зібрати Zebra, скористайтеся такими командами:

   __cargo build --release__

   Цей процес може зайняти деякий час. Переконайтеся, що ваш Raspberry Pi достатньо охолоджується, оскільки компіляція може спричиняти нагрівання.

5. __Конфігурація__
   Створіть файл конфігурації для Zebra. Ви можете використати конфігурацію за замовчуванням як відправну точку:

   __cp zcash.conf.example zcash.conf__

   Відредагуйте файл zcash.conf, щоб налаштувати параметри вашого вузла. Ви можете вказати мережу, увімкнути майнінг, налаштувати з’єднання з іншими вузлами тощо.

6. __Запустіть Zebra__
   Тепер ви можете запустити Zebra зі своєю користувацькою конфігурацією:

   __./target/release/zebrad -c zcash.conf__

   __git comment__ 

   Ця команда запустить вузол Zebra, і він почне синхронізацію з блокчейном Zcash.

7. __Моніторинг__
   Ви можете відстежувати прогрес і стан вашого вузла Zebra, відкривши веббраузер і перейшовши за адресою __http://127.0.0.1:8233/status__.

## Усунення несправностей

Якщо у вас виникнуть будь-які проблеми зі збиранням або запуском Zebra, перегляньте [документацію Zebra](https://doc.zebra.zfnd.org/docs/intro.html), щоб знайти поради з усунення несправностей і додаткову інформацію.

Обов’язково стежте, щоб ваш Raspberry Pi не перегрівався, оскільки запуск вузла може спричиняти нагрівання. Ви можете скористатися системою охолодження, наприклад вентилятором або радіатором.

## Висновок

Дотримуючись цього посібника, ви маєте успішно налаштувати й запустити Zebra на своєму Raspberry Pi 4. Тепер ви робите внесок у мережу Zcash як незалежний вузол, допомагаючи захищати приватність транзакцій Zcash.

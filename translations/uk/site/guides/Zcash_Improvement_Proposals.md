# Що таке ZIP
Пропозиція щодо вдосконалення Zcash (ZIP) — це проєктний документ, який надає інформацію спільноті Zcash або описує нову функцію для Zcash, його процесів чи середовища. ZIP має містити стислу технічну специфікацію функції та обґрунтування цієї функції.

Ми розглядаємо ZIP як основний механізм для пропонування нових функцій, збору відгуків спільноти щодо певного питання та документування проєктних рішень, що були закладені в Zcash. Власник(и) ZIP (зазвичай автор(и)) відповідає за досягнення консенсусу в межах спільноти та документування незгодних думок.

Оскільки ZIP підтримуються як текстові файли у версійному репозиторії, історія їхніх змін є історичним записом пропозиції щодо функції.

-> [zips.z.cash](https://zips.z.cash)


## Редактори ZIP

Кожен ZIP, що надходить, має пройти процес рецензування. Редактори ZIP відповідають за те, щоб формат і зміст будь-якого ZIP належним чином описували терміни, обґрунтування та внесені технічні зміни. 

```markdown
* Jack Grigg, Kris Nuttycombe and Daira-Emma Hopwood associated with the ECC.
* Conrado, and Arya, associated with the ZF.
* Sam and Mark, associated with SL.
```

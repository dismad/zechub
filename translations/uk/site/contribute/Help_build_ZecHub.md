# Допоможіть розбудовувати ZecHub 

Якщо ви є учасником спільноти Zcash і хочете зробити внесок у розбудову ZecHub, є кілька речей, які будуть надзвичайно корисними.

Добирайте посилання для новинної розсилки та додавайте їх на GitHub-сторінку розсилки: -> [Натисніть тут, щоб переглянути інструкції](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Перевіряйте факти у документах вікі, пропонуйте правки та нові сторінки на Github

Створюйте відеоконтент у таких категоріях:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Створюйте дизайни, такі як постери/графіка/анімації, для екосистеми Zcash і Privacy 

Перекладайте будь-які з наявних сторінок вікі.

Повідомлення про помилки

#### Щоб дізнатися подробиці про суми винагород, будь ласка, прочитайте нашу сторінку для учасників -> [тут](https://zechub.wiki/contribute/contributing-guide#content).

Щопонеділка ми публікуємо задачі, за які наразі відкриті винагороди. Ви також можете пропонувати винагороди! Переважно їх можна знайти на ZEC Bounties.

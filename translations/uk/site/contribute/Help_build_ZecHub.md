# Допоможіть розбудовувати ZecHub 

Якщо ви є учасником спільноти Zcash і хочете долучитися до розбудови ZecHub, є кілька речей, у яких ваша допомога буде надзвичайно корисною.

Добирайте посилання для інформаційного бюлетеня та додавайте їх на GitHub-сторінку бюлетеня: -> [Натисніть тут для інструкцій](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Перевіряйте факти у документації вікі, пропонуйте правки та нові сторінки на Github

Створюйте відеоконтент у таких категоріях:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Створюйте дизайни, такі як постери/графіка/анімації для екосистеми Zcash і Privacy 

Перекладайте будь-які наявні сторінки вікі.

Повідомлення про помилки

#### Щоб дізнатися подробиці про розміри винагород, будь ласка, прочитайте нашу сторінку про внесок -> [тут](https://zechub.wiki/contribute/contributing-guide#content).

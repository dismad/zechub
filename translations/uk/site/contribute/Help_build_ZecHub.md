# Допоможіть розбудовувати ZecHub

Якщо ви є учасником спільноти Zcash і хочете долучитися до розбудови ZecHub, є кілька речей, які будуть надзвичайно корисними.

Добирайте посилання для розсилки та додавайте їх на GitHub-сторінку розсилки: -> [Натисніть тут, щоб переглянути інструкції](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Перевіряйте факти у вікі-документації, пропонуйте правки та нові сторінки в Github

Створюйте відеоконтент у таких категоріях:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Створюйте дизайни, такі як постери/графіка/анімації для екосистеми Zcash та Privacy

Перекладайте будь-які з наявних сторінок вікі.

Повідомлення про помилки

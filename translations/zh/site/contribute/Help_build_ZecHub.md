# 帮助建设 ZecHub

如果你是 Zcash 社区成员，并且想为建设 ZecHub 作出贡献，以下几项会非常有帮助。

为新闻通讯整理链接，并将它们添加到新闻通讯的 GitHub 页面：-> [点击这里查看说明](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

核查 wiki 文档中的事实，提出编辑建议，并在 Github 中提议新页面

创建以下类别的视频内容：
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

为 Zcash 和隐私生态制作海报/图形/动画等设计作品

翻译任何现有的 wiki 页面。

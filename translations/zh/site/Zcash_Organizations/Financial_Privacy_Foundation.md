# Financial Privacy Foundation (FPF)
![FPF](https://i.ibb.co/BsZc8Qv/FPFgrants.webp)

## 关于 FPF
Financial Privacy Foundation (FPF) 是一家注册于开曼群岛的非营利组织。FPF 的主要目标是教育并赋能个人和社区，使其能够掌控自己的金融隐私与安全。FPF 还致力于推动建立一个公平、公正并尊重个人隐私的金融体系。FPF 在金融隐私相关领域的主要活动，是为个人和社区提供信息、资源与支持，以保护并优化他们的金融隐私。Financial Privacy Fund (FPF) 目前正在创建一项名为 FPF X FPF 的资助计划。

##  关于 FPF X FPF 的信息

 FPF X FPF 是 FPF 的首个实验性资助计划，即 Financial Privacy Fund (FPF x FPF)。该计划将资助 Zcash 生态系统内接近完成的工程项目，或能够立即为生态系统做出贡献，并可在规定时间和资金限制内完成的项目。FPF X FPF 在启动时资金非常有限，旨在完善资助流程，并希望在未来推出后续轮次。 


## [FPF x ZecHub 赏金计划 - 博客文章](https://www.financialprivacyfoundation.org/post/partnership-announcement-zechub)

## FPF X FPF 计划详情 

FPF X FPF 计划详情 FPF 的计划包含以下几个要点：

- 资助金额：最高 $50,000 USD
- 单项最高申请金额：$25,000 USD
- KYC 要求：无
- 项目类型：仅考虑能够直接惠及 Zcash 生态系统并且可以完整交付的工程项目。
- 付款方式：项目成功完成后支付
- 启动日期：2024 年 3 月 18 日

## 资助申请流程 

FPF X FPF 的资助申请流程包含多个环节。首先，申请人必须通过提供的安全表单提交申请。随后，FPF 将审核申请是否符合资格，申请人将在申请截止日期后的 5 个日历日内收到状态通知。接下来，一个临时评审委员会将在提交截止日期后的 20 个日历日内审核并决定资助授予结果。为保证透明度和问责性，已授予资助的详细信息将公布在 FPF 网站上。

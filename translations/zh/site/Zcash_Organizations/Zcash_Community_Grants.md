# <img src="https://i.ibb.co/bjYy15Yb/ZCG-Circle-Logo-10.png" alt="替代文本" width="50"/>  Zcash 社区资助

## 使命宣言

Zcash Community Grants 项目为进入 Zcash 生态系统的独立团队提供资金支持，以开展面向 Zcash 生态系统公共利益的重要持续开发（或其他工作）。

## 人员与团队

Zcash Community Grants 由一个委员会管理，目前由五人组成。该社区负责审查并批准希望基于 Zcash 协议开展建设的第三方团队的资助申请。更多信息请参见[团队](https://zcashcommunitygrants.org/committee/)

近期经 ZCG 批准的项目包括由 QEDIT 团队主导的 Zcash Shielded Assets (ZSA)。该项目通过一套新的支付协议将 DeFi 引入 ZEC 和 Zcash，为 Zcash 主网增加更多功能。

Zcash Media 的短纪录片邀请了 Edward Snowden、Zooko 和 Deirdre Connolly 等知名 Zcash 社区人士参与，旨在制作具有相关性、娱乐性和教育意义、并能被广泛传播的视频。

Ledger 的硬件钱包集成已实现对 Sapling 屏蔽交易的完整支持，这使得 Zcash 可以被私密且安全地存储。该工作由 Zondax 团队完成。

ZGo 正在开发一套适用于实体商店的简易一键式屏蔽支付和销售点系统 SDK。

Global Ambassador 项目帮助 Zcash 在国际范围内获得更广泛的代表性，并支持全球各地的线下社区活动。

Zcash Grants Committee 还审查了更多项目，请查看已提交项目的[完整更新列表](https://docs.google.com/spreadsheets/d/1FQ28rDCyRW0TiNxrm3rgD8ai2KGUsXAjPieQmI1kKKg/edit?gid=135980745#gid=135980745)。

## 组织结构

Zcash Community Grants 是一个独立于 Electric Coin Company 和 Zcash Foundation 的组织。

## 资金来源

Electric Coin Company 获得[Zcash development fund](https://zips.z.cash/zip-1014) 的 8%。他们使用这笔资金来支持第三方开发团队以及内部运营。

### 第 1 步：准备你的提案
创建一份清晰的提案，包括：
- 项目标题和申请人详情
- 问题陈述和拟议解决方案
- 交付成果、时间线和里程碑
- 预算和资金申请
- 相关经验与风险

---

### 第 2 步：在 GitHub 上提交
1. 前往 **Zcash Community Grants GitHub 仓库**
2. 打开 **Issues** 标签页并点击 **New Issue**
3. 选择合适的**资助申请模板**
4. 填写所有必填字段并提交

GitHub issue 是**正式申请记录**。

---

### 第 3 步：创建社区论坛帖子
1. 前往 **Zcash Community Forum**
2. 在 **Zcash Community Grants** 分类中创建一个新主题
3. 使用与 GitHub issue 相同的标题
4. 添加简短摘要，并附上 **GitHub issue 链接**

---

### 第 4 步：参与评审
- 关注 GitHub issue 和论坛主题
- 回应来自 ZCG Committee 和社区的问题与反馈

---

### 第 5 步：结果
- 获批的提案将进入资助和报告阶段
- 未获批准的提案**可以**修改后重新提交

---

**注意：** GitHub 是正式记录来源；论坛用于公开讨论。

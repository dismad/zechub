# 什么是 ZIP
Zcash 改进提案（ZIP）是一种设计文档，用于向 Zcash 社区提供信息，或描述 Zcash、其流程或其环境中的新功能。ZIP 应当提供该功能的简明技术规范，以及该功能的设计原理说明。

我们希望 ZIP 成为提出新功能、收集社区对某个问题意见，以及记录已纳入 Zcash 的设计决策的主要机制。ZIP 的所有者（通常是作者）负责在社区内建立共识，并记录不同意见。

由于 ZIP 以文本文件形式维护在版本化仓库中，其修订历史就是该功能提案的历史记录。

-> [zips.z.cash](https://zips.z.cash)


## ZIP 编辑者

每一个提交的 ZIP 都必须经过审查流程。ZIP 编辑者的职责是确保任何 ZIP 的格式和内容都能充分描述相关术语、设计原理以及所做的技术变更。 

```markdown
* Jack Grigg, Kris Nuttycombe and Daira-Emma Hopwood associated with the ECC.
* Conrado, and Arya, associated with the ZF.
* Sam and Mark, associated with SL.
```

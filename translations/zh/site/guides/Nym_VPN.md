# Nym VPN

Nym 是一种不断演进的混合网络（mixnet），这是一类以隐私为目标的计算机网络基础设施，通过分离源 IP 地址和目标 IP 地址来隐藏用户元数据。它可匿名化多种类型的通信，包括消息传递、文件传输、支付交易以及对基础网站的网页浏览。该项目基于免费且开源的软件构建，并且是去中心化的，由全球分布的一组独立节点共同维护。Nym VPN 与 Mixnet 接受受屏蔽的 ZEC 支付！

### 访问受地理限制的内容

绕过地理限制，访问你所在地区不可用的流媒体服务、网站或内容。

### 提升你在公共 WiFi 上的安全性

在咖啡馆、机场或酒店使用公共 Wi-Fi 时，保护你的数据免受黑客攻击。

### 一体化高级隐私保护

保护你的隐私  
加密你的互联网连接，向 ISP、广告商和数据追踪器隐藏你的在线活动。

### 在线匿名

隐藏你的 IP 地址，在浏览时保持匿名，避免被画像分析或接收定向广告。

### 避免带宽限速

防止 ISP 在你进行流媒体播放或游戏时故意降低你的网络连接速度。

### 防止监控与审查

绕过审查，在限制性国家访问被屏蔽的网站或社交媒体。

### 保障网上银行安全

在访问网上银行或进行支付时，保护敏感的金融信息。

### 避免动态定价

隐藏你的位置，避免在购买航班、酒店或商品时因地理位置而遭受价格歧视。

### 保护你的加密货币交易

借助 NymVPN 的不可关联支付和高级元数据保护，保护你的加密货币活动。

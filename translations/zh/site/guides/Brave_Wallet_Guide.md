# 如何使用 Brave Wallet 发送和接收屏蔽式 Zcash (ZEC)：一步步掌握隐私交易

Brave Wallet 是一款安全的原生加密钱包，直接内置于 Brave 浏览器中，无需安装扩展程序。


## 为什么支持屏蔽式 ZEC 很重要


Zcash 是少数默认提供隐私保护并支持屏蔽交易的加密货币之一。

Brave Wallet 现已支持 Orchard 屏蔽式 ZEC，使用户无需借助第三方软件即可进行私密交易。

现在，用户可以在 Brave 浏览器中发送和接收 ZEC，而无需暴露余额或元数据。

## 第 1 部分：设置 Brave Wallet

**涵盖步骤：**

1. 打开 Brave 浏览器（目前仅支持桌面端）

2. 点击钱包图标，或前往 brave://wallet

3. 设置你的钱包（创建或导入）

4. 进入加密资产仪表板


![img1](https://github.com/user-attachments/assets/f54cd1a1-8569-4925-ba1c-7597d030593e)


## 第 2 部分：在 Brave Wallet 中添加 Zcash (ZEC)

**涵盖步骤：**

1. 在钱包仪表板中，点击“Manage Assets”

2. 搜索 ZEC 并启用

3. 现在 ZEC 会显示在你的钱包中


![img2](https://github.com/user-attachments/assets/6f2b2190-cf55-4394-9d5f-29ff9b5bb525)


## 第 3 部分：接收屏蔽式 ZEC（Unified Address）

**涵盖步骤：**

1. 在你的资产中点击 ZEC

2. 选择“Receive”

3. Brave 会生成一个 Unified Address (UA) —— 一个同时支持屏蔽交易和透明交易的地址

4. 复制地址或扫描二维码

5. 将你的 UA 分享给发送方



![img3](https://github.com/user-attachments/assets/53c940b6-1a03-4fa7-aefa-d3478f678a88)



## 第 4 部分：私密发送屏蔽式 ZEC

**涵盖步骤：**

1. 选择 ZEC -> Send

2. 粘贴收款方的 Unified Address 或屏蔽地址

3. 输入金额

4. 确认交易详情

5. 点击 Send —— 资金现已通过 Orchard 资金池私密转账


![img4](https://raw.githubusercontent.com/Kellyjoe8/zechub/refs/heads/main/Internet_20250808_172118_4.webp)




## 地址类型

以下是对每种 Zcash 地址类型的说明：

**t-addr (t1...)** - 透明地址的工作方式类似于 Bitcoin 地址，这意味着交易在区块链上是公开的。它们更容易与非 Zcash 钱包配合使用，但不提供隐私保护。

**z-addr (zs...)** - 屏蔽地址使用 Zcash 的隐私技术（Sprout/Sapling）来隐藏发送方、接收方和金额。只有 Viewing Key 或所有者才能披露相关细节。

**UA (u... or QR)** - Unified Addresses 将多种地址类型（透明、屏蔽、Orchard）合并为一个地址。在尽可能优先保护隐私的同时，也实现了最大的兼容性。

# 使用 IPFS 和 Brave Browser 进行 P2P 文件共享

我们可以使用 IPFS 以点对点的方式在彼此之间共享大型媒体文件。

那么问题是什么：IPFS 网关不适用于大文件，它们会超时。

**要求**：[IPFS Desktop](https://docs.ipfs.tech/install/ipfs-desktop/) + [Brave Browser](https://brave.com/)

简而言之，视频在最后。

我们可以在 ZF AV Club 中用这个来传输大型媒体文件，并把它记录下来分享给大家，因为这很有趣，而且超级酷！！

*另外*，我们将直接与 Protocol Labs 的 IPFS 团队合作，开发这个使用场景。
这也是在 dWeb Camp 建立联系后带来的另一个好处！

如果接收文件的设备没有安装 IPFS，且使用的浏览器不是 Brave：你将需要等待网关加载文件。*对于大文件，这会返回 504 错误。*

接下来：如何在你的机器上固定其他人的文件，这样当原始节点离线时，我们就不会遇到单点故障问题。

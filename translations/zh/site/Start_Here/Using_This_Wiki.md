# 使用本 Wiki

其中一些内容可能会让人不知所措。我们理解！这里有很多信息。下面是一份简单的分步指南，帮助你了解应该如何浏览这个 wiki。

## 刚接触 Zcash

如果你刚接触 Zcash，这里是一个很好的入门起点。你首先应该查看以下页面：

[Zcash 新用户指南](https://zechub.wiki/start-here/new-user-guide#content)

[Zcash 资源](https://zechub.wiki/start-here/zcash-resources#content)

[Zcash 基础知识](https://zechub.wiki/start-here/what-is-zec-and-zcash#content)

## 准备开始使用 Zcash

在你阅读完介绍文档及其相关资源后，可以查看 [使用 ZEC](https://zechub.wiki/guides/using-zec-privately#content) 和 [教程](https://youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O&si=4vleWUbCGbNJu1-M) 视频，开始了解如何使用 ZEC。

## 我是 Zcash 社区成员

如果你是社区成员，那太好了！想要及时了解最新动态，最好的方式就是关注每周通讯 [Zcash 生态摘要](https://zechub.substack.com/s/zcash-ecosystem-digest) 以及 [Zcash 社区页面列表](https://zechub.wiki/zcash-community/community-links#content) 中的所有账号。

## 我想为 ZecHub 做贡献

太好了！请先阅读 [什么是 ZecHub](https://zechub.wiki/start-here/what-is-zechub#content)，然后查看 [帮助建设 ZecHub](https://zechub.wiki/contribute/help-build-zechub) 以了解如何参与贡献。

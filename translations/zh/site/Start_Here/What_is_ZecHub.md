# 什么是 ZecHub？
---

ZecHub 是一个面向 Zcash 的去中心化教育中心。[ZecHub](http://zechub.xyz) 的目标是提供一个教育平台，让社区成员能够协作创建、验证并推广支持 Zcash 生态系统的内容。我们通过以下几种方式来实现这一点。

1. [教程](https://www.youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O)
2. [Wiki 文档](https://zechub.wiki/)
3. [采访社区成员的播客](https://www.youtube.com/playlist?list=PL6_epn0lASLHlNCMtUErX8UfaJK6N9K5O)。
4. [每周 Zcash 新闻简报。](https://zechub.substack.com/)

这些教程、短篇博客和新闻简报都是开源并托管在 GitHub 上的，这意味着社区中的任何人都可以参与贡献。这样做是为了确保内容准确，同时让该渠道本身不受单点故障影响。

如果你是 Zcash 新手，请使用[本指南](/site/Start_Here/New_User_Guide)。****

# Zcash 基础知识

## ZEC 用例

#### 支付

人们使用 ZEC 以低手续费高效且安全地进行交易。屏蔽的 Zcash 可确保交易保持机密性，同时也让他们可以在自愿的情况下选择共享地址和交易信息。私密的 “Z-Z” 交易会出现在公共区块链上，因此外界知道这些交易确实发生过且手续费已支付。但地址、交易金额和备注字段都经过加密，公众无法看到。

#### 价值储存

人们也可以使用 ZEC 将财富储存在一种稀缺且保护隐私的资产中。ZEC 的总量将永远只有 2100 万枚，这意味着该资产具有固定供应量。一旦第 2100 万枚 ZEC 被[挖出并进入流通](https://z.cash/mining-zcash/)，该资产将具备抗通胀特性。在中心化把关者增发国家货币供应的情况下，抗通胀资产是对冲通胀的良好工具。

#### 消息传递

人们可以使用屏蔽 Z-Z 交易中的备注字段发送完全加密的消息。要发送备注，用户必须发起一笔交易并支付交易手续费，费用低于 $0.01。

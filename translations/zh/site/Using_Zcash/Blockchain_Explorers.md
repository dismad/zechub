# 区块链浏览器

## 什么是浏览器？

区块链浏览器本质上是一种工具，用于将区块链上已经发生的内容可视化。它们接收以下输入：交易 ID、地址或区块哈希，并以可视化方式展示发生了什么。

## Zcash 区块链浏览器列表：

- [Zcash 区块链浏览器](https://mainnet.zcashexplorer.app/)

- [Blockchair](https://blockchair.com)

- [3xpl](https://3xpl.com/zcash)

- [Bitquery](https://explorer.bitquery.io/zcash)

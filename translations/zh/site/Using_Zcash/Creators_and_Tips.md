## Zcash.me

![Zcash.me 标志](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### 概述

**Zcash.me** 是一个个人资料目录与发现平台，让任何人都能轻松找到 Zcash 用户并向其发送 ZEC。它相当于“Zcash 版 Linktree”——用一个简单链接分享你的屏蔽地址和社交身份。

| 项目              | 详情 |
|-------------------|---------|
| **网站**       | [https://zcash.me/](https://zcash.me/) |
| **Twitter**       | [@zcashme](https://x.com/zcashme) |
| **资料格式**| `zcash.me/[username]` |
| **状态**        | 已上线（部分功能仍在开发中） |

### 主要功能

#### 当前可用
- **自定义个人资料页面** - 包含用户名、简介、头像和所在地的个性化页面
- **屏蔽地址展示** - 为你的统一地址提供一键复制按钮
- **社交链接集成** - 支持 X、LinkedIn、Telegram 和自定义网站
- **验证系统** - 通过已验证徽章建立社区信任
- **推荐排行榜** - 跟踪顶级推荐者和社区领袖
- **活动状态** - 显示上次活跃时间

#### 即将推出
| 功能            | 描述 |
|--------------------|-------------|
| 位置地图       | 查找附近的 Zcash 用户 |
| 平台统计| 查看增长指标 |
| 社区投票    | 对生态主题进行投票 |
| 新闻与公告 | 官方更新 |
| 论坛              | 社区讨论 |
| 移动应用         | 完整的移动端体验 |

### 快速开始
1. 访问 [https://zcash.me/](https://zcash.me/) 并点击 **Join**
2. 选择一个唯一用户名，添加简介和头像
3. 粘贴你的 Zcash 统一地址（`u1...`）
4. 连接你的社交账号
5. 分享你的链接：`zcash.me/[yourusername]`

## TipZ

![TipZ 标志](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### 概述

**TipZ** 是一个专为创作者和 cypherpunks 打造的**私密打赏协议**。粉丝可以用任何主流加密货币打赏；系统会自动将其转换并以**屏蔽 ZEC**的形式交付。

| 项目              | 详情 |
|-------------------|---------|
| **网站**       | [https://tipz.cash/](https://tipz.cash/) |
| **Twitter**       | [@tipz_cash](https://x.com/tipz_cash) |
| **资料格式**| `tipz.cash/@[username]` |
| **状态**        | 已上线 |

> *“tipz.cash 是一个面向创作者和 cypherpunks 的私密打赏协议。”*

### 为什么隐私对创作者很重要

#### 财务自由
- 0% 平台手续费
- 非托管与自托管
- 无中间人

#### 隐私优先设计
- 每一笔打赏都会以屏蔽 ZEC 形式到账
- 私密的钱包余额
- 随打赏附带加密消息
- 提供创作者完全匿名选项

#### 跨链支付
| 货币 | 网络 |
|----------|---------|
| ETH      | Ethereum |
| USDC     | 多条链 |
| SOL      | Solana |
| ZEC      | 直接通过 Zcash |

#### 创作者保护
- 无需担心被平台封禁
- 抗审查
- 无需 KYC

# 水龙头

<img width="2138" height="878" alt="截图_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />


水龙头是免费发放少量加密货币的服务。开发者通常用它来试验加密货币，用户也可以在无需购买的情况下获得少量加密货币。它们通常会限制用户在特定时间内可领取的加密货币数量。

主网：<a href="https://zecfaucet.com">zecfaucet</a>

测试网：<a href="https://testnet.zecfaucet.com">testnet.zecaucet.com</a>

测试网：<a href="https://fauzec.com/">fauzec.com</a>

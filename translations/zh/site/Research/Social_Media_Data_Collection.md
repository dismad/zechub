# 社交媒体数据收集

目前，很难找到一个不处于社交网络影响范围之内的人。社交网络因其易用性，以及在传播信息方面的速度和效率而广受欢迎。

不幸的是，同样的便利性也让我们作为用户，对社交媒体平台收集了我们多少信息、它们能用这些信息做什么，以及如果恶意第三方获取这些信息后可能会做什么，变得不够警惕。

接下来，我们将概述其中一些平台会收集我们的哪些数据、它们会如何使用这些数据，以及为什么这可能对我们的隐私，甚至潜在地对我们个人或家庭的安全构成威胁。

Discord
-------

我们需要先从 Discord 说起，因为这是我们“家里”使用的通信应用。

Discord 在管理围绕各种主题的大型社区方面非常有用，但和许多其他平台一样，它也同样受益于我们的个人信息。

### Discord 存储的数据

根据其隐私政策，Discord 会收集我们的以下信息：

-   文本消息（包括草稿）、语音消息、直播封面图片、你添加的好友或与你互动的用户、你玩的游戏、你注册加入的服务器或社区、你通过该应用进行的购买或销售，以及你在内容审核时作出的决定。

他们不会存储你的视频或你的直播视频，尽管他们也为未来改变这一决定留下了余地。

他们还会自动收集某些信息，例如你的连接 IP 地址、你使用的浏览器以及你的操作系统。

### 他们如何使用这些数据

他们使用我们数据所做的大多数事情都可以被视为有合理性，比如改善用户体验、提供更加个性化的体验，以及维护社区标准。不过，他们也可以利用你的数据来推广自己的服务，甚至可能与第三方共享其中一部分数据，以推广其服务或分析公司的增长情况。

他们还声称，可以在任何机构通过法院命令提出要求时，使用这些信息并将其提供给相关机构。

Twitter
-------

和 Discord 一样，Twitter 也会在其服务器上存储相当多的信息。它还会保存私信、你关注的人，以及你互动过的人（保存的帖子、收藏），实际上，它们的政策非常相似，包括与第三方和相关机构共享你的数据。

Meta
----

Meta 旗下的应用程序，包括 Facebook、Instagram，以及现在的 Threads，做得更进一步。仅你主动提供的信息，其列表就已经长到难以一一详述。其中包括你的所有帖子，包括语音笔记、使用你设备“相机”功能或相册拍摄的照片、你看到的广告、你花了多少时间浏览这些广告，甚至还包括那些你选择忽略的广告。还包括你消息中的元数据，例如位置、日期、时间和设备信息。它还包括你在应用各项不同功能上花费的时间；如果法律没有保护你，还可能包括你的宗教信仰、哲学观点、性取向，甚至工会成员身份。

但他们还更进一步，通过著名的 Pixel，Meta 能够知道你在 **Facebook 之外** 做了什么：你访问了哪些网站、在这些网站上停留了多久、你玩了什么游戏、你买了什么，甚至还包括你的教育程度等数据。当然，也包括你设备的技术信息，例如 IP 地址、操作系统、浏览器等等，无论你是否登录过 Facebook。

为什么我应该在意？
------------------

“我没什么可隐瞒的”是人们淡化这些平台收集我们所有数据重要性时最常见的论点。然而，即使你所做的事情并不违法，仍然有充分的理由去担心这些公司究竟了解你多少。

如果一位政治活动人士所反对的某个强大组织认定其行为是“错误的”，并向这些公司中的任何一家索取此人的信息，那么这位活动人士可能会受到严重影响，[而这些公司会毫不犹豫地提供这些信息](https://gizmodo.com/twitter-saudi-arabia-x-jack-dorsey-areej-al-sadhan-abdu-1850805085)。

也有很多明确的例子表明，你的个人信息最终可能落入陌生人、公司或黑客手中，而他们可能会将其用于恶意目的，例如身份盗窃、获取你的账户访问权限、发送垃圾广告、勒索你，或者[操纵你](https://en.wikipedia.org/wiki/Facebook%E2%80%93Cambridge_Analytica_data_scandal)。

同样地，你也可能失去对自己数据的控制。你可能并不知道你的数据是如何被存储、处理或与第三方共享的；如果你改变主意，也可能无法删除或修改这些数据。如果你所信任的服务提供商[没有妥善保护你的数据](https://www.iotworldtoday.com/security/duolingo-data-breach-exposes-3-million-user-emails) [正确地](https://edition.cnn.com/2023/01/05/tech/twitter-data-email-addresses/index.html)，这同样会让你面临风险。

有没有替代方案？
-----------------------

虽然它们没有那么流行，但至少对于大多数最受欢迎的社交网络来说，都存在替代方案。我们来看几个例子：

### Matrix，像 Discord 一样但更注重隐私

Matrix 是最接近 Discord 和 Telegram 的替代方案，拥有它们的许多功能，同时增加了隐私增强特性，例如端到端加密、不把数据当作生意来做，以及保持代码开源。如果你决定运行自己的服务器，它是完全免费的，不过你也可能希望为互联网上的专业服务器付费，或者向该项目捐款。它支持跨平台，甚至允许与 Discord、Telegram 以及其他流行应用集成。

### Mastodon

这是最受欢迎的 Twitter 替代方案之一，尽管并不是唯一的。Mastodon 的突出特点在于它是去中心化的，这意味着任何愿意的个人或组织都可以启动自己的实例或“服务器”，供具有共同兴趣或特征的人使用，但他们仍然可以与其他独立实例互动。

在 Mastodon 中，每位管理员都会制定自己的规则，而且由于他们不从事数据生意，因此不会收集过多关于用户、人口统计特征或偏好的信息，也不会将这些信息分享给第三方。

不幸的是，他们并未在消息中实现端到端加密，这一点需要牢记。

### Signal

正如我们在消息客户端 wiki 中详细介绍的那样，Signal 是 Telegram 和 WhatsApp 的一个更加私密且更安全的替代方案，同时也具备许多非常有吸引力且有趣的功能

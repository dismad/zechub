# 零知识资产平台



**[Firn Protocol](https://app.firn.cash/)**：Firn 是首个基于账户模型的零知识隐私平台，并为基于 Ethereum 的链引入了可插拔、灵活的隐私功能。利用零知识证明，Firn 为 Ethereum 和基于 Ethereum 的 L2 用户提供安全、高效的资金隐私。**它如何运作？**
要使用 Firn，请先将 ETH 存入该协议。一旦你拥有 Firn 余额，就可以私密地向其他 Firn 用户转账，或与其他协议（如 Uniswap）交互。你也可以私密地将资金提回网络。Firn 对所有 ETH 提现收取 0.79% 的小额费用。这些费用将按比例分配给 FIRN Token 持有者 - [白皮书](https://firn.cash/whitepaper.pdf) 


**[RAILGUN](https://railgun.org/):** 作为一个 layer-1 智能合约，RAILGUN 是一个私密转账和 DeFi 基础设施，其存在依托于 Ethereum、Polygon、Binance Smart Chain 和 Arbitrum。
设置你的非托管 RAILGUN Wallet，将任意 ERC-20 token 屏蔽到你选择的 0zk 地址中；一旦完成屏蔽，token、余额和交易都会被加密  - [白皮书](https://docs.railgun.org/wiki) 


**[StarkEX](https://starkware.co/starkex/):** 一个已在 Ethereum Mainnet 上线的 Layer-2 扩容引擎。交易会在链下批量处理并发送到 StarEX 服务。StarEX 验证交易并更新相关余额；它会基于交易的有效性生成 STARK 证明。链上验证器智能合约会接收并验证该 STARK 证明 - [白皮书](https://docs.starkware.co/starkex-v4/)


**[Manta Network](https://www.manta.network/):** 面向 Web 3、DeFi 等场景的链上隐私。Manta Atlantic 是最快的 ZK L1 链，通过合规的链上隐私和身份功能，为 web3 带来可编程隐私。$MANTA 的固定总供应量为 1,000,000,000，没有通胀计划。$MANTA token 在被赎回后将立即自动销毁 - [白皮书](https://docs.manta.network/) 

**[Boltz](https://boltz.exchange/):**  Boltz 是一个 Layer 2 解决方案。Boltz 是一个以隐私为先、非托管的 bitcoin 交易平台，旨在连接 Bitcoin 的不同层，例如 liquid 和 lightning network。借助 Boltz，用户可以在不同层之间无缝交换他们的 bitcoin。Boltz Swaps 是非托管的，这意味着用户在整个交换流程中始终可以完全掌控自己的 bitcoin  [白皮书](https://docs.boltz.exchange/en/latest/) 


**[ShadeProtocol](https://shadeprotocol.io/)**：Shade Protocol 是构建在 Secret Network 上的一组相互连接、保护隐私的 layer-2 DeFi 应用。这些核心应用包括稳定币、治理、债券、质押衍生品、保险、合成资产、借贷、DEX 等  - [白皮书]

**[PantherProtocol](https://www.pantherprotocol.io/)**：一个零知识、跨协议层，用于保护你的链上数据并实现合规的 DeFi 访问。**它如何运作？** 将你的资产存入 Panther，以使用 1:1 抵押的屏蔽资产 zAssets。在 Panther Pools 中存储多种资产类型，同时用零知识证明保护你的链上数据。面对众多 Layer-1 和 2 区块链，它们各自提出不同的价值、理念和机制，Panther 致力于构建基础设施，以私密方式将它们连接起来，而不是期待其中某一条链接管一切。 - [白皮书](https://docs.pantherprotocol.io/)

**[Light Shield](https://shield.lightprotocol.com/)**：Light 是一个开源的 layer 2 zkLayer，可实现私密程序执行，专为 Solana 打造。Light Protocol 允许链上状态被加密。只有用户自己可以解密其私有状态；因此他们实质上“拥有”它。该私有状态可以由单个用户拥有，也可以由一组共享解密密钥的用户共同拥有。
例如，考虑 Solana 上的一个 NFT 挂单；其公开状态（如价格）对所有人可见，包括它正在出售这一事实。但出价和交易应当保持私密 - **资产兑换**：是 - [白皮书](https://shield.lightprotocol.com/) - ![Light Shield 标志](https://miro.medium.com/v2/resize:fit:1358/1*C4Pe23afCwS05seCeZWBLA.png)

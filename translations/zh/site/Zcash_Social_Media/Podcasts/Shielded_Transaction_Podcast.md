# Forsight Institute 的 Shielded Transactions 播客

Foresight Institute 是一家研究机构和非营利组织，致力于支持高影响力技术的良性发展。自 1987 年以“引导强大技术”为愿景创立以来，我们不断发展，成为一个多元化组织，专注于若干传统机构因目标过于宏大而难以支持的科学与技术领域。

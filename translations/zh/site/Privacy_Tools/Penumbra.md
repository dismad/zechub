# Penumbra 

Penumbra 旨在成为 Cosmos 生态中具有竞争力的、最前沿的全隐私 layer-1 网络。它使用户能够在不暴露链上元数据的情况下，安全地进行交易、质押、代币交换和做市。作为一条完全隐私保护的区块链，它还支持在 Web 上进行隐私交易，同时不牺牲隐私性或去中心化——Penumbra 默认启用隐私，不存在透明交易，也不存在透明价值池。

## Penumbra IBC 中使用的技术 

IBC 集成：排在首位的是 Inter-Blockchain Communication (IBC) 协议，这是一项突破性且创新的技术，为 Penumbra 的运行提供支持。它对于提升网络内的互操作性至关重要。IBC 还通过让不同的区块链网络能够通信和交换数据来发挥作用。此外，IBC 在促进 Penumbra 之间的无缝资产转移、数据共享和交互方面也扮演着关键角色。这种互操作性和技术帮助用户轻松地在各类区块链之间转移代币和资产。另一方面，IBC 也有助于扩展性，能够提升交易速度、优化性能，并减少 Penumbra 网络中的延迟。

零知识证明（ZKPs）：Penumbra 利用零知识证明密码学技术，确保所有交易都能在不泄露任何细节的情况下完成验证和确认，例如发送方、接收方或转账金额。

具备隐私性的去中心化交易所：隐私是 Panumbra 网络的核心。该网络可能采用一种去中心化交易机制，在保护金融数据的同时增强交易和加密货币交易的隐私，并赋予用户控制其交易策略披露范围的权利。

## 使用场景

安全交易：Penumbra 允许用户使用任何加密资产体验隐私交易。由于 Penumbra 是一条完全隐私保护的区块链，它允许用户以私密方式进行交易，在跨链流转时保护发送方、接收方、金额和资产类型；它确保交易保持私密，未经用户许可无法被追踪或解密。值得一提的是，每个 Penumbra 地址都支持查看功能——这一设计允许用户选择性地向其授权的第三方披露活动信息，以满足合规或其他需求。有趣的是，得益于权益证明共识机制，Penumbra 交易速度极快且手续费较低。 

质押：从宏观角度来看，Penumbra 希望始终处于领先地位。作为世界上首个隐私保护的权益证明网络，它的核心目标是确保委托人的隐私以及验证者的责任性。Penumbra 提供的另一项有趣功能是 PEN 质押代币，它在保护网络安全的同时，也赋予代币持有者通过链上投票流程参与项目治理的权利。网络中的每位委托者都会获得一种名为 dPEN 的原生代币。需要说明的是，dPEN 记录在隐私池中，并且可以像其他任何资产一样进行交易。从链上治理的角度来看，网络的未来掌握在利益相关者手中，他们通过对提案进行投票并参与决策，确保项目不被中心化实体控制。 

代币交换：代币交换是 Penumbra 的另一项使用场景。该网络支持任意类型加密资产之间的隐私交换——一经提交，所有交换都会被密封！通过利用 ZSwap，用户可以交换任意加密资产交易对，而无需暴露交易历史或时间戳等敏感细节，这在一定程度上有助于打击金融欺诈。虽然 Maximal Extractable Value (MEV) 常见于透明链，但像 Penumbra 这样的全隐私链不存在 MEV，因为网络交易不会泄露有关用户活动的数据。

做市：作为一条完全隐私保护的链，Penumbra 让做市商能够自主控制是否披露其交易策略和特征相关信息，而不是将其全部活动暴露给所有人。

# Web Browsers

1. **Brave：** 当前市场上最知名且最安全的浏览器之一。它会拦截侵犯隐私的广告和追踪器，并阻止第三方数据存储。

2. **Mulvad：** 一款以隐私为核心的网页浏览器，由 Mullvad VPN 与 Tor Project 合作开发。其设计目标是尽可能减少追踪和指纹识别。

3. **Tor：** 这款浏览器会阻止 Flash、RealPlayer、QuickTime 等插件，因为它们可能被操控以泄露你的 IP 地址。我们不建议在 Tor Browser 中安装额外的附加组件或插件。


## 选择网页浏览器时有哪些安全注意事项？

在选择网页浏览器时，有几点需要牢记：安全漏洞和利用方式通常会被持续发现。有些浏览器在某种程度上是可追踪的，但也有一些浏览器允许用户匿名浏览——例如 [Tor browser](https://www.torproject.org/download/)。

**指纹识别**：如果你希望保护诸如设备上已安装的软件、硬件配置等隐私信息，请考虑选择支持抗指纹识别功能的网页浏览器。

**HTTPS 测试**：选择网页浏览器时，另一个需要关注的安全因素是 HTTPS Test 协议，它有助于加密网站连接，并隐藏 cookies、URL 及其他类型的敏感元数据。使用 HTTPS 时，连接会被加密，网络中的任何第三方都无法读取服务器与你的浏览器之间传输的内容。

**追踪 Cookie 防护**：此外，全球互联网上有很大一部分网页都隐藏着第三方追踪器；这些追踪器会在你的浏览器中读取和写入 cookies。这些 cookies 可能构成恶意威胁，因为它们可被用于跨网站追踪你的浏览行为；为防止这种追踪，请选择能够拦截未知浏览 cookies 的网页浏览器。

**导航测试**：当你从一个网页跳转到另一个网页时，会遇到许多链接（有些是站内链接，有些则跳转到其他网站）。当你点击这些链接时，某些浏览器 API 会允许第一个网站与第二个网站通信。为解决这一问题，请选择能够在你点击链接时阻止网站共享追踪数据的网页浏览器。在选择你认为适合自身需求的浏览器时，请确保它具备强大的隐私功能，以及内置广告拦截器等关键特性。此外，许多这类网页浏览器支持扩展和附加组件，而这些扩展有时可能带来安全风险，因此请务必始终从安全的生态系统或可信来源安装这些扩展。
 
## Brave Browser Zcash Wallet Integration

Brave 浏览器正在为 Web3 生态系统开辟道路。如何做到的？如今，超过 200 万个 unstoppable domains 已可用于展示去中心化网站。凭借 Brave Browser 强大的安全特性，人们可以在不被追踪的情况下探索互联网。Brave 与 Zcash 正在携手，为 Web3 生态系统带来新的隐私工具。为了获得更大收益，Brave 将通过其内置加密钱包快速集成 Zcash 协议，从而革新加密货币交易——这一集成将使用户能够非常轻松且安全地存储、发送和接收 ZEC，以及其他加密货币。Brave 网页浏览器与 Zcash 的合作将真正改变格局，因为用户将对自己的 Web3 活动拥有更大的控制权。

关于 Brave 浏览器与 Zcash 集成，另一个很有意思的事实是其私密且去中心化的消息功能；Brave、Electric Coin Company 以及 Filecoin Foundation 团队正在为 Brave 浏览器铺设以隐私为核心的功能——此次合作旨在通过 Zcash 协议提供私密消息和媒体传输。在这里，Interplanetary File System (IPFS) 将在为加密内容提供安全存储能力方面发挥关键作用。

# Graphene OS

GrapheneOS 是一个非营利、开源项目，致力于在保持与 Android 应用兼容的同时，提升移动设备的隐私与安全性。这个操作系统非常重视推进隐私和安全技术，尤其专注于增强沙盒机制、漏洞利用缓解措施，以及改进权限模型。它诞生于 2014 年，当时名为 **CopperheadOS**。此后，它已发展为守护你数字隐私与安全的强大屏障。

GrapheneOS 的主要目标是在不影响用户体验的前提下，加强诸如应用沙盒等安全边界。

GrapheneOS 可能会为特定功能引入多种开关，例如网络权限、传感器权限，或设备锁定时的限制（涵盖 USB 外设、相机访问和快捷设置磁贴等方面）。此外，它还精心设计了更复杂、面向用户的隐私与安全功能，在保持易用体验的同时，为每项功能配备相应的界面增强。

## **GrapheneOS 的功能**

**安全加固** GrapheneOS 包含广泛的安全增强功能，例如采用内存安全编程语言以及基于编译器的安全特性，以减少常见漏洞。

**沙盒机制** 它强化了应用沙盒隔离，使应用彼此隔离，从而限制潜在攻击路径。

**已验证启动** GrapheneOS 使用硬件支持的密钥和可验证的启动流程，以确保操作系统的完整性。

**增强权限控制** 它提供了更完善的应用权限控制，让用户能够精细调整和管理应用可访问的数据。

**隐私仪表板** 用户可以通过隐私仪表板监控和控制应用行为，从而更透明地了解数据使用情况。

**内置安全更新** GrapheneOS 提供及时的安全更新，确保用户获得针对漏洞的最新保护。

**强加密** 它默认采用全磁盘加密，保护设备上存储的数据安全。

**改进的浏览器安全性** 默认浏览器经过增强安全配置，其中包括防跟踪保护。

**最少预装应用** GrapheneOS 仅预装极少量应用，从而降低潜在的安全和隐私风险。

**USB 敌对端口防护** 当设备锁定时，它可防止通过 USB 端口对设备进行未授权访问。

**反恶意软件保护** GrapheneOS 包含用于检测和阻止已知恶意软件的功能。

**以隐私为核心** 该系统在设计上高度重视用户隐私，尽可能减少数据收集和暴露。

**开源** 它是一个开源项目，具备透明性，并允许社区参与以持续改进安全性。

**可自定义的安全策略** 用户可以根据自己的偏好自定义各种安全策略。

**兼容性** GrapheneOS 致力于保持与 Android 应用的兼容性，使用户在享受增强的安全与隐私功能的同时，继续使用自己喜爱的应用程序。

## **安装 GrapheneOS**

如最佳实践部分所述，建议使用官方推荐的安装指南。安装 GrapheneOS 有两种方式：使用基于 [WebUSB 的安装器](https://grapheneos.org/install/web)，或使用[命令行安装指南](https://grapheneos.org/install/cli)

**基于 Web USB 的安装** 若要使用 Web 安装器方式安装 GrapheneOS，通常需要以下材料和资源：

- 2GB 可用内存和 32GB 可用存储空间。
- USB 数据线（A 或 C）
- 支持的操作系统：Windows 10、Windows 11、macOS Big Sur（11 - 13）、Arch Linux、Debian（10 - 12）、Ubuntu（22.04、22.10 和 23.04）、ChromeOS、GrapheneOS、Google Android（原生 Pixel OS）
- 支持的浏览器：Chromium（Ubuntu 不受支持）、Vanadium（GrapheneOS）、Google Chrome、Microsoft Edge、Brave Browser。

请通过下方链接查看安装指南，并按照安装流程进行操作

[GrapheneOS Web 安装器指南](https://grapheneos.org/install/web#prerequisites)

**命令行安装** 不建议不具备技术背景的用户使用命令行安装，且命令行安装需要以下工具和资源；

- 你应至少拥有 2GB 可用内存和 32GB 可用存储空间
- USB 数据线（A 或 C）
- 支持的操作系统 Windows 10 Windows 11 macOS Big Sur (11) macOS Monterey (12) macOS Ventura (13) Arch Linux Debian 10 (buster) Debian 11 (bullseye) Debian 12 (bookworm) Ubuntu 20.04 LTS Ubuntu 22.04 LTS Ubuntu 22.10 Ubuntu 23.04
- 通过 Android、ChromeOS 或 GrapheneOS 使用 Web 安装器

请通过下方链接查看安装指南，并按照安装流程进行操作

[命令行安装](https://grapheneos.org/install/cli#prerequisites)

## **最佳实践**

**保持系统为最新版本** 定期更新 GrapheneOS，以确保你获得最新的安全补丁和增强功能。

**使用强密码** 为设备加密和应用登录设置强且唯一的密码，以防止未授权访问。

**应用权限** 仔细审查和管理应用权限，只授予每个应用正常运行所必需的权限。

**定期备份** 定期备份你的数据，以防设备丢失或数据损坏。

**加密存储** 如果默认未启用，请加密你的设备存储，以便在设备丢失或被盗时保护数据。

**安全锁屏** 使用安全的锁屏方式，例如 PIN、密码或生物识别认证，以防止未授权访问。

**避免 Root 你的 Android 设备** 避免 Root 或解锁引导加载程序，因为这会削弱设备安全性。

**验证应用来源** 验证应用及其来源的真实性，以避免安装恶意软件。

**安装注重隐私的浏览器** 可考虑使用注重隐私的浏览器，如 Brave browser、Firefox 或 Bromite，以获得更安全的浏览体验。

**定期审计应用** 定期检查并卸载你不再使用或不再信任的应用，以减少攻击面。

**启用双因素认证 (2FA)** 为你的在线账户启用 2FA，以增加额外一层安全防护。

**避免使用公共 Wi-Fi** 连接公共 Wi-Fi 网络时请保持谨慎，因为它们通常不够安全。必要时请使用 VPN。

**谨慎处理位置数据** 限制应用的位置追踪；如果你重视隐私，也可以考虑使用位置伪装工具。

**避免未知链接和附件** 对未经请求的链接和电子邮件附件保持警惕，因为它们可能是网络钓鱼尝试或恶意软件。

**检查默认设置** 仔细查看并调整默认设置，使其符合你的隐私偏好。

**社区支持** 积极参与 GrapheneOS 社区和论坛，以获取技巧、更新和安全建议。你可以点击[这里](https://grapheneos.org/contact)联系团队并保持最新动态，探索更多联系方式。

**使用官方安装方式** GrapheneOS 提供两种官方支持的安装方式。用户可以选择基于 WebUSB 的安装器，这是对大多数人推荐的方法；也可以遵循面向技术能力更强用户的命令行安装指南。

## **结论**

归根结底，GrapheneOS 的核心在于提供一种以用户隐私和安全为优先的替代型移动操作系统，让用户在其数字生活中拥有更大的控制权，同时仍可兼容他们所依赖的应用程序。在隐私与安全问题日益重要的时代，它是一个致力于为移动设备安全和隐私树立高标准的项目。

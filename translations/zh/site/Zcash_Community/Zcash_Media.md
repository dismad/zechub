# Zcash Media

“我们在这项协作努力中的角色，是为当前和未来的 Zcashers 提供教育、启发并激发热情，建立一个可与最大型项目比肩甚至超越它们的支持者基础。”

## 背景

Zcash Media 的教育视频由 [37 LAINES](https://www.37laines.com/) 为公共利益制作，内容聚焦 Zcash 和加密货币。它由 [Zcash Community Grants](https://x.com/ZcashCommGrants) 资助，并已成为 Zcash 生态系统中的重要参与者。

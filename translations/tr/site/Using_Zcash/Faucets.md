# Faucet'ler

<img width="2138" height="878" alt="Screenshot_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />


Faucet'ler, küçük miktarlarda kripto parayı ücretsiz olarak dağıtan hizmetlerdir. Genellikle geliştiriciler tarafından kripto para ile deneyler yapmak için, kullanıcılar tarafından ise satın almak zorunda kalmadan küçük miktarlarda kripto para elde etmek için kullanılırlar. Tipik olarak, bir kullanıcının belirli bir zaman dilimi içinde alabileceği kripto para miktarını sınırlarlar.

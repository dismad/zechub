# 개발자 자료

[Zcash Docs](https://zcash.readthedocs.io/en/latest/)

[Mastering Bitcoin: Programming The Open Blockchain](https://www.amazon.com/Mastering-Bitcoin-Programming-Open-Blockchain-ebook/dp/B071K7FCD4)

[Zcashd RPCs](https://zcash.github.io/rpc/)

[The Zcashd Book](https://zcash.github.io/zcash/)

[The Zebrad Book](https://zebra.zfnd.org/)

이 목록은 Zcash Foundation 및 ECC에서 개발자들이 제공한 추천 사항을 바탕으로 작성되었습니다.

추가 자료(링크, 책, 채널 등)로 목록을 업데이트할 것이 있다면 언제든지 포함해 주세요!

*팀은 모든 것을 완전히 최신 상태로 유지하는 데 필요한 자원이 항상 있는 것은 아니지만, 마주치는 문제를 기록하면 가이드 및 자료의 방향성을 지시하게 됩니다.*

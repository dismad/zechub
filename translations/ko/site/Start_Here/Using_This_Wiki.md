# 이 위키 사용법

이 내용 중 일부는 압도적으로 느껴질 수 있습니다. 이해합니다! 여기에 많은 정보가 포함되어 있기 때문입니다. 아래 단계별 가이드를 통해 위키를 탐색하는 방법을 쉽게 확인할 수 있습니다.

## Zcash 초보자

Zcash에 새로 오셨다면, 이곳에서 기본적인 내용을 배우기에 적합한 곳입니다. 먼저 다음 페이지들을 확인하시기 바랍니다:

[Zcash 신규 사용자 가이드](https://zechub.wiki/start-here/new-user-guide#content)

[Zcash 자료](https://zechub.wiki/start-here/zcash-resources#content)

[Zcash 기초 지식](https://zechub.wiki/start-here/what-is-zec-and-zcash#content)

## Zcash 사용 준비 완료

소개 문서와 관련 자료를 읽으셨다면, [ZEC 사용하기](https://zechub.wiki/guides/using-zec-privately#content) 및 [튜토리얼](https://youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O&si=4vleWUbCGbNJu1-M) 영상들을 확인하여 ZEC 사용법을 배우시기 바랍니다.

## 저는 Zcash 커뮤니티 구성원입니다

커뮤니티 구성원이신다면, 훌륭합니다! 최신 정보를 얻으려면 [Zcash Ecosystem Digest](https://zechub.substack.com/s/zcash-ecosystem-digest) 주간 뉴스레터를 구독하시고, [Zcash 커뮤니티 페이지 목록](https://zechub.wiki/zcash-community/community-links#content)에 있는 모든 계정을 팔로우하시기 바랍니다.

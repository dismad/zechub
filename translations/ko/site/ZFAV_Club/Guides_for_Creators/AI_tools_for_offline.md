### 이미지 생성

이미지 생성을 위한 모델은 어디에서 다운로드할 수 있나요?
[**HuggingFace**](https://huggingface.co/models?pipeline_tag=text-to-image&sort=trending) 및 [**Civitai**](https://civitai.com/)에는 다양한 모델이 있습니다.

## Windows - 커스텀 모델 다운로드

일반적으로 AI 도구를 사용하려면 최소 4GB 또는 이상의 GPU 메모리가 필요합니다.

Nvidia GPU는 가장 잘 작동합니다.

- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (4GB GPU 메모리 필요)
- [NMKD Stable Diffusion GUI](https://nmkd.itch.io/t2i-gui)
- [EasyDiffusion 3.0](https://github.com/easydiffusion/easydiffusion)

### Mac OS - 커스텀 모델 다운로드

- AppStore에서 무료로 제공되는 [Draw Things: AI Generation](https://apps.apple.com/ee/app/draw-things-ai-generation/id6444050820) (여러 가지 다른 StableDiffusion 모델 사용 가능)
- 무료로 제공되는 [DiffusionBee](https://diffusionbee.com/) (여러 가지 다른 StableDiffusion 모델 사용 가능)
- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (4GB GPU 메모리 필요)

### Linux - 커스텀 모델 다운로드

- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (4GB GPU 메모리 필요)

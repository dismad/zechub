# Graphene OS

GrapheneOS는 비영리, 오픈소스 프로젝트로서 모바일 기기에서의 개인정보 보호와 보안을 강화하면서도 Android 앱과 호환성을 유지하는 것을 목표로 합니다. 이 운영체제는 특히 샌드박싱, 공격 방어 기법, 권한 관리 모델 개선에 중점을 두고 있는 보안 및 개인정보 보호 기술의 발전에 집중하고 있습니다. 2014년 **CopperheadOS**라는 이름으로 탄생했으며 이후로는 귀하의 디지털 개인정보와 보안을 지키는 강력한 수호자로 성장했습니다.

GrapheneOS의 주요 목적은 사용자 경험에 부정적인 영향을 미치지 않으면서도 앱 샌드박싱과 같은 보안 경계를 강화하는 것입니다. 

GrapheneOS에서는 네트워크 권한, 센서 권한 또는 기기 잠금 상태일 때의 제한(USB 주변 장치, 카메라 접근 및 빠른 타일 등)과 같은 특정 기능에 대한 다양한 토글을 도입할 수 있습니다. 또한 사용자에게 보다 복잡한 개인정보 보호 및 보안 기능은 사용자 친화적인 경험을 유지하면서도 각각의 사용자 인터페이스 개선과 함께 세심하게 설계되었습니다.

## **GrapheneOS의 주요 특징**

**보안 강화** GrapheneOS는 메모리 안전 프로그래밍 언어 및 컴파일러 기반 보안 기능을 포함하여 일반적인 취약점을 줄이는 방식으로 광범위한 보안 개선을 제공합니다.

**샌드박싱** 앱 간 격리를 강화하여 잠재적 공격 벡터를 제한하는 앱 샌드박싱을 강화합니다.

**인증 부팅(Verified Boot)** GrapheneOS는 하드웨어 기반의 키와 검증 가능한 부팅 프로세스를 사용하여 운영체제의 무결성을 보장합니다.

**개선된 권한 관리** 앱에 대한 접근 데이터를 세부적으로 조정하고 관리할 수 있는 개선된 권한 제어 기능을 제공합니다.

**개인정보 대시보드** 사용자는 개인정보 대시보드를 통해 앱의 행동을 모니터링하고 통제하여 데이터 사용에 대한 투명성을 확보할 수 있습니다.

**내장 보안 업데이트** GrapheneOS는 최신 보안 패치로 사용자가 취약점으로부터 보호되도록 정기적으로 제공되는 보안 업데이트를 포함합니다.

**강력한 암호화** 기본값으로 전체 디스크 암호화를 적용하여 기기 내 저장된 데이터를 보호합니다.

**개선된 브라우저 보안** 기본 브라우저는 추적 방지 등 보안을 강화하도록 구성되어 있습니다.

**최소한의 사전 설치 앱** GrapheneOS는 보안 및 개인정보 유출 위험을 줄이기 위해 최소한의 사전 설치 앱만 포함합니다.

**USB 포트 보호 기능** 기기가 잠겨 있을 때 USB 포트를 통해 무단 접근을 방지하는 기능을 제공합니다.

**악성코드 방어 기능** GrapheneOS는 알려진 악성코드를 탐지하고 예방하기 위한 기능을 포함합니다.

**개인정보 보호 중심 설계** OS는 사용자 개인정보 보호에 중점을 두고 데이터 수집 및 노출을 최소화하도록 설계되었습니다.

**오픈소스** 이 프로젝트는 오픈소스로, 보안 향상을 위한 커뮤니티 기여와 투명성을 가능하게 합니다.

**사용자 맞춤형 보안 정책** 사용자는 자신의 선호에 따라 다양한 보안 정책을 설정할 수 있습니다.

**호환성** GrapheneOS는 Android 앱과의 호환성을 유지하려고 노력하여, 사용자가 즐겨 쓰는 앱들을 계속 사용하면서도 강화된 보안 및 개인정보 보호 기능을 누릴 수 있도록 합니다.

## **GrapheneOS 설치**

최선의 실천 방법 섹션에서 언급했듯이 공식적으로 권장되는 설치 가이드를 따르는 것이 좋습니다. GrapheneOS 설치에는 두 가지 방법이 있으며, 하나는 [WebUSB 기반 설치자](https://grapheneos.org/install/web)를 사용하는 것, 다른 하나는 [명령줄 설치 가이드](https://grapheneos.org/install/cli)를 사용하는 것입니다.

**웹 USB 기반 설치** 웹 인스톨러 방법을 사용하여 GrapheneOS를 설치하려면 일반적으로 다음과 같은 자원과 물품이 필요합니다:

- 2GB의 자유 메모리 및 32GB의 자유 저장 공간.
- USB 케이블(A 또는 C)
- 지원되는 운영체제: Windows 10, Windows 11, macOS Big Sur (11 - 13), Arch Linux, Debian (10 - 12), Ubuntu (22.04, 22.10 및 23.04), ChromeOS, GrapheneOS, Google Android (스토크 Pixel OS)
- 지원되는 브라우저: Chromium(ubuntu에서는 지원되지 않음), Vanadium(GrapheneOS), Google Chrome, Microsoft Edge, Brave Browser.

아래 링크를 통해 설치 가이드를 확인하고 설치 절차를 따르세요

[GrapheneOS 웹 인스톨러 가이드](https://grapheneos.org/install/web#prerequisites)

**명령줄 설치** 명령줄 설치는 기술적으로 익숙하지 않은 사용자에게는 권장되지 않으며, 아래 도구 및 자원이 명령줄 설치에 필요합니다.

- 최소 2GB의 자유 메모리와 32GB의 자유 저장 공간을 확보해야 합니다.
- USB 케이블(A 또는 C)
- 지원되는 운영체제: Windows 10, Windows 11, macOS Big Sur (11), macOS Monterey (12), macOS Ventura (13), Arch Linux, Debian 10 (buster), Debian 11 (bullseye), Debian 12 (bookworm), Ubuntu 20.04 LTS, Ubuntu 22.04 LTS, Ubuntu 22.10, Ubuntu 23.04
- Android, ChromeOS 또는 GrapheneOS에서의 웹 인스톨러

아래 링크를 통해 설치 가이드를 확인하고 설치 절차를 따르세요 

[명령줄 설치](https://grapheneos.org/install/cli#prerequisites)

## **최선의 실천 방법**

**운영체제 업데이트 유지** GrapheneOS를 정기적으로 업데이트하여 최신 보안 패치와 개선 사항을 확보하세요.

**강력한 비밀번호 사용** 기기 암호화 및 앱 로그인에 강하고 고유한 비밀번호를 설정하여 무단 접근을 방지하세요.

**앱 권한 관리** 각 앱이 정상적으로 작동하기 위해 필요한 최소한의 권한만 부여하고, 주의 깊게 검토 및 관리하세요.

**정기 백업 수행** 기기 손실 또는 데이터 손상 시 대비하여 데이터를 정기적으로 백업하세요.

**저장소 암호화** 기본값이 아닌 경우, 기기 저장소를 암호화하여 기기가 분실되거나 도난될 경우 데이터 보호에 유리합니다.

**보안 잠금 화면 사용** PIN, 비밀번호 또는 생체 인증과 같은 보안 잠금 화면을 사용하여 무단 접근을 방지하세요.

**안드로이드 기기 루팅 피하기** 기기를 루팅하거나 부트로더를 해제하는 것은 기기의 보안을 약화시킬 수 있으므로 피해야 합니다.

**앱 출처 확인** 앱과 그 출처의 진위를 확인하여 악성 소프트웨어 설치를 방지하세요.

**개인정보 중심 브라우저 사용** Brave 브라우저, Firefox 또는 Bromite와 같은 개인정보 보호에 중점을 두는 브라우저를 사용하여 안전한 브라우징을 고려하세요.

**앱 정기 감사** 더 이상 사용하거나 신뢰하지 않는 앱을 주기적으로 검토하고 제거하여 공격 표면을 줄이세요.

**2단계 인증(2FA) 활성화** 온라인 계정에 2FA를 활성화하여 보안의 추가 레이어를 제공하세요.

**공공 Wi-Fi 피하기** 공공 Wi-Fi 네트워크에 연결할 때 주의해야 하며, 필요 시 VPN을 사용하세요.

**위치 데이터 조심하기** 위치 추적을 제한하고 개인정보 보호가 중요하다면 위치 위장 도구를 고려하세요.

**알 수 없는 링크 및 첨부 파일 피하기** 요청되지 않은 링크와 이메일 첨부 파일에 주의해야 하며, 이는 피싱 시도 또는 악성코드일 수 있습니다.

**기본 설정 검토** 기본 설정을 신중하게 검토하고 조정하여 개인정보 선호도에 맞게 설정하세요.

**커뮤니티 지원 활용** GrapheneOS 커뮤니티 및 포럼에서 팁, 업데이트 및 보안 조언을 얻으세요. 팀과 연락하거나 더 많은 연락처를 확인하려면 [여기](https://grapheneos.org/contact)를 클릭하세요.

**공식 설치 방법 사용** GrapheneOS는 두 가지 공식적으로 지원되는 설치 방법을 제공합니다. 대부분의 개인에게 추천되는 웹 USB 기반 인스톨러를 선택하거나, 더 기술적인 사용자에게 맞춤화된 명령줄 설치 가이드를 따르는 것이 가능합니다.

## **결론**

요약하자면, GrapheneOS는 사용자의 개인정보와 보안을 우선시하는 대체 모바일 운영체제를 제공하며, 사용자가 의존하는 앱과의 호환성을 유지하면서도 디지털 생활에 대한 더 많은 통제권을 부여합니다. 이 프로젝트는 개인정보 및 보안이 중요한 시대에 모바일 기기의 보안과 개인정보 보호 기준을 높이는 것을 목표로 합니다.

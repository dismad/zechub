# Zcash 결제 처리기

## CoinPayments
- **지원 유형**: 투명  
- **설명**: 100개 이상의 암호화폐를 지원하는 글로벌 암호화폐 결제 게이트웨이로, Zcash도 포함됩니다. 온라인 상점을 위한 간단한 통합을 제공합니다.  
- **URL**: [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="CoinPayments 로고" width="200"/>

---

## CipherPay
- **지원 유형**: 가리킴  
- **설명**: 몇 분만에 Zcash를 수락할 수 있으며, 비유지보유식이며, 구매자 데이터가 없으며, 중개자가 없습니다.  
- **URL**: [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="CipherPay 로고" width="200"/>

---

## Flexa
- **지원 유형**: 투명  
- **설명**: Flexa는 상인들이 일상적인 거래에 암호화폐를 사용할 수 있도록 하는 암호화폐 결제 처리기입니다. Zcash(ZEC)도 포함됩니다.  
- **URL**: [Flexa](https://flexa.co/)

## BTCPay Server
- **지원 유형**: 가리킴

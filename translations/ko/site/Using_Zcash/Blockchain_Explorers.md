# 블록체인 탐색기

## 탐색기란 무엇인가?

블록체인 탐색기는 단순히 블록체인에서 이미 발생한 일들을 시각화할 수 있는 도구입니다. 입력값으로는 거래 ID, 주소 또는 블록 해시를 사용하고, 이를 통해 어떤 일이 발생했는지 시각적으로 보여줍니다.

## Zcash 블록 탐색기 목록:

- [Zcash 블록 탐색기](https://mainnet.zcashexplorer.app/)

- [Blockchair](https://blockchair.com)

- [3xpl](https://3xpl.com/zcash)

- [Bitquery](https://explorer.bitquery.io/zcash)

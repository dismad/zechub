# Faucets

<img width="2138" height="878" alt="Screenshot_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />

 Faucets는 소량의 암호화폐를 무료로 제공하는 서비스입니다. 개발자가 암호화폐를 실험하거나, 사용자가 구매하지 않고도 암호화폐를 조금씩 얻고자 할 때 자주 사용됩니다. 일반적으로 faucets는 특정 시간 동안 사용자가 받을 수 있는 암호화폐의 양에 제한이 있습니다.

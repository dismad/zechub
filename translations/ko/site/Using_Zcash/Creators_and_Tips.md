## Zcash.me

![Zcash.me 로고](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### 개요

**Zcash.me**는 누구나 Zcash 사용자에게 ZEC를 쉽게 찾고 보낼 수 있는 프로필 디렉토리 및 발견 플랫폼입니다. 이는 "Zcash의 Linktree"라고 할 수 있으며, 단일 링크로 쉴드 주소와 소셜 존재감을 공유하는 간단한 방법입니다.

| 항목              | 세부 정보 |
|-------------------|---------|
| **웹사이트**       | [https://zcash.me/](https://zcash.me/) |
| **트위터**        | [@zcashme](https://x.com/zcashme) |
| **프로필 형식**   | `zcash.me/[유저명]` |
| **상태**          | 운영 중 (기능 개발 중) |

### 주요 기능

#### 현재 제공
- **커스텀 프로필 페이지** - 사용자 이름, 자기소개, 아바타 및 위치를 포함한 개인화된 페이지
- **쉴드 주소 표시** - 통합 주소 복사 버튼 (하나 클릭)
- **소셜 링크 통합** - X, LinkedIn, Telegram 및 커스텀 웹사이트
- **인증 시스템** - 커뮤니티 신뢰를 위한 인증 배지
- **추천 리더보드** - 최고 추천자와 커뮤니티 리더 추적
- **활동 상태** - 마지막 활동 시간 표시

#### 곧 출시 예정
| 기능            | 설명 |
|------------------|------|
| 위치 지도        | 근처 Zcash 사용자 찾기 |
| 플랫폼 통계      | 성장 메트릭 보기 |
| 커뮤니티 투표    | 생태계 주제에 대한 투표 |
| 뉴스 및 공지     | 공식 업데이트 |
| 포럼             | 커뮤니티 토론 |
| 모바일 앱        | 완전한 모바일 경험 |

### 시작 방법
1. [https://zcash.me/](https://zcash.me/)에 방문하고 **Join**을 클릭합니다.
2. 고유한 사용자 이름을 선택하고 자기소개 및 프로필 사진을 추가합니다.
3. Zcash 통합 주소(`u1...`)를 붙여넣습니다.
4. 소셜 계정을 연결합니다.
5. 링크를 공유: `zcash.me/[당신의 사용자명]`

## TipZ

![TipZ 로고](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### 개요

**TipZ**는 크리에이터와 사이퍼펑크를 위한 **개인정보 보호 기능을 갖춘 팁 프로토콜**입니다. 팬들은 주요 암호화폐로 팁을 보내면 자동으로 **쉴드 ZEC**로 전환되어 전달됩니다.

| 항목              | 세부 정보 |
|-------------------|---------|
| **웹사이트**       | [https://tipz.cash/](https://tipz.cash/) |
| **트위터**        | [@tipz_cash](https://x.com/tipz_cash) |
| **프로필 형식**   | `tipz.cash/@[유저명]` |
| **상태**          | 운영 중 |

> *"tipz.cash는 크리에이터와 사이퍼펑크를 위한 개인정보 보호 기능을 갖춘 팁 프로토콜입니다."*

### 크리에이터에게 중요한 개인정보 보호

#### 재정적 자유
- 플랫폼 수수료 0%
- 비소유자 관리 및 자가 소유
- 중개자 없음

#### 개인정보 보호 중심 설계
- 모든 팁은 쉴드 ZEC로 도착
- 개인 지갑 잔액
- 팁과 함께 암호화된 메시지
- 크리에이터의 완전한 익명 선택 가능

#### 크로스 체인 결제
| 통화 | 네트워크 |
|------|---------|
| ETH  | 이더리움 |
| USDC | 여러 체인 |
| SOL  | 솔라나 |
| ZEC  | 직접 Zcash |

#### 크리에이터 보호
- 플랫폼에서 제거되는 위험 없음
- 검열 저항성
- KYC 필요 없음

### 추가 자료

```
TipZ 결제 흐름

팬 -> tipz.cash 프로토콜 -> 크리에이터의 지갑 (쉴드 ZEC)

팬이 금액 + 메시지를 선택 -> 프로토콜이 자동으로 전환 -> 크리에이터가 개인적인 쉴드 ZEC를 받음
```

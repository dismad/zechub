# ZIP이란 무엇인가?
Zcash 개선 제안(ZIP)은 Zcash 커뮤니티에 정보를 제공하거나, Zcash 및 그 프로세스나 환경에 대한 새로운 기능을 설명하는 설계 문서입니다. ZIP은 기능의 간결한 기술적 명세와 기능 도입의 이유를 제시해야 합니다.

ZIP은 새 기능을 제안하고, 특정 문제에 대해 커뮤니티의 의견을 수집하며, Zcash에서 내려진 설계 결정을 문서화하는 주요 수단이 되어야 합니다. ZIP의 소유자(보통 작성자)는 커뮤니티 내 합의를 도출하고, 반대되는 의견들을 기록하는 책임이 있습니다.

ZIP은 버전 관리된 저장소에 텍스트 파일로 유지되므로, 그들의 수정 이력은 기능 제안의 역사적 기록이 됩니다.

-> [zips.z.cash](https://zips.z.cash)


# ZecHub 구축을 도와주세요

당신이 Zcash 커뮤니티의 일원이며 ZecHub 구축에 기여하고 싶다면, 매우 도움이 되는 몇 가지 방법이 있습니다.

뉴스레터용 링크를 큐레이션하고 뉴스레터의 GitHub 페이지에 추가하세요: -> [지침을 보려면 여기를 클릭하세요](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

위키 문서를 사실 확인하고, 편집을 제안하며, Github에서 새 페이지를 제안하세요

다음 카테고리의 비디오 콘텐츠를 제작하세요:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Zcash 및 Privacy 생태계를 위한 포스터/그래픽/애니메이션 등의 디자인을 제작하세요

기존 위키 페이지를 번역하세요.

버그 신고

#### 바운티 금액에 대한 자세한 내용은 기여 페이지를 읽어주세요 -> [여기](https://zechub.wiki/contribute/contributing-guide#content).

# ZecHub 기여하기

Zcash 커뮤니티의 일원이며, ZecHub를 구축하는 데 기여하고자 한다면 몇 가지 매우 유용한 방법이 있습니다.

뉴스레터에 링크를 정리하고 GitHub 페이지에 추가하세요: -> [지침 보기](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

위키 문서의 사실 확인, 편집 제안 및 새로운 페이지 제안

다음 범주에서 영상 콘텐츠를 생성하세요:
```
* Zcash 설명 영상
* Zcash 지갑 가이드/튜토리얼
* 제3자 애플리케이션 시연
```

Zcash 및 프라이버시 생태계를 위한 포스터/그래픽/애니메이션 디자인을 생성하세요

기존 위키 페이지의 번역을 수행하세요.

버그 신고

#### 보상 금액에 대한 자세한 내용은 기여 페이지에서 확인해 주세요 -> [여기](https://zechub.wiki/contribute/contributing-guide#content).

# ZecHub 구축을 도와주세요

여러분이 Zcash 커뮤니티의 일원이며 ZecHub 구축에 기여하고 싶다면, 매우 큰 도움이 되는 몇 가지 방법이 있습니다.

뉴스레터용 링크를 큐레이션하여 뉴스레터의 GitHub 페이지에 추가해 주세요: -> [지침을 보려면 여기를 클릭하세요](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

위키 문서를 팩트 체크하고, 수정 사항을 제안하며, Github에서 새 페이지를 제안해 주세요.

다음 카테고리의 영상 콘텐츠를 만들어 주세요:
```
* Zcash 설명 영상
* Zcash 지갑 가이드/튜토리얼
* 서드파티 애플리케이션 데모
```

Zcash 및 Privacy 생태계를 위한 포스터/그래픽/애니메이션 등의 디자인을 제작해 주세요.

기존 위키 페이지 중 어떤 것이든 번역해 주세요.

버그 신고

#### 바운티 금액에 대한 자세한 내용은 기여 페이지를 읽어 주세요 -> [여기](https://zechub.wiki/contribute/contributing-guide#content).

현재 바운티가 열려 있는 작업에 대한 이슈는 매주 월요일에 게시합니다. 바운티를 제안할 수도 있습니다! 주로 ZEC Bounties에서 확인할 수 있습니다.

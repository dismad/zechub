# Web Browsers

1. **Brave:** आज के बाज़ार में सबसे प्रमुख और सुरक्षित ब्राउज़रों में से एक। यह गोपनीयता में दखल देने वाले विज्ञापनों और trackers को ब्लॉक करता है। यह third-party data storage को ब्लॉक करता है। 

2. **Mulvad:** गोपनीयता-केंद्रित web browser, जिसे Mullvad VPN और Tor Project के सहयोग से विकसित किया गया है। इसे tracking और fingerprinting को न्यूनतम करने के लिए बनाया गया है

3. **Tor:** यह browser Flash, RealPlayer, QuickTime और अन्य जैसे plugins को ब्लॉक करता है: इन्हें इस तरह से प्रभावित किया जा सकता है कि वे आपका IP address उजागर कर दें। हम Tor Browser में अतिरिक्त add-ons या plugins इंस्टॉल करने की अनुशंसा नहीं करते


## Web Browser चुनते समय सुरक्षा संबंधी कौन-सी बातों पर विचार करना चाहिए?

Web browser चुनते समय कुछ बातों को ध्यान में रखना चाहिए: सुरक्षा कमजोरियाँ और exploits आमतौर पर नियमित रूप से खोजे जाते हैं। कुछ browsers किसी न किसी रूप में trace किए जा सकते हैं, लेकिन ऐसे browsers भी हैं जो उपयोगकर्ताओं को गुमनाम रूप से browse करने की अनुमति देते हैं - उदाहरण के लिए [Tor browser](https://www.torproject.org/download/)। 

**Fingerprinting**: यदि आप अपने device पर इंस्टॉल किए गए softwares, hardware configuration और अन्य चीज़ों को निजी रखना चाहते हैं, तो ऐसे web browser पर ध्यान दें जो fingerprinting resistance feature का समर्थन करता हो। 

**HTTPS Test**: Web browser चुनते समय एक और सुरक्षा संबंधी बात HTTPS Test protocol है, जो website connection को encrypt करने में मदद करता है और cookies, URLs तथा अन्य प्रकार के संवेदनशील metadata को छिपाता है। HTTPS के साथ, connection encrypt हो जाता है और network पर कोई भी third party server और आपके browser के बीच transfer हो रही सामग्री को नहीं पढ़ सकती। 

**Tracking Cookie Protection**: इसके अलावा, world wide web के web pages का एक बड़ा हिस्सा छिपे हुए third-party trackers रखता है; ये trackers आपके browser में cookies को पढ़ते और लिखते हैं। ये cookies हानिकारक जोखिम पैदा कर सकती हैं क्योंकि इनका उपयोग विभिन्न websites पर आपकी browsing को track करने के लिए किया जा सकता है; इस tracking को रोकने के लिए ऐसा web browser चुनें जो अज्ञात browsing cookies को ब्लॉक करता हो।

**Navigation Test**: जब आप एक web page से दूसरे पर जाते हैं, तो आपको कई links मिलेंगे (कुछ internal links होते हैं जबकि कुछ किसी दूसरी website पर ले जाते हैं)। जब आप इन links पर click करते हैं, तो कुछ browser APIs पहली site को दूसरी site के साथ communicate करने की अनुमति देते हैं। इसका समाधान करने के लिए ऐसा web browser चुनें जो link पर click करने पर websites को tracking data साझा करने से रोके। जिस browser को आप अपनी ज़रूरत के लिए उपयुक्त समझते हैं, उसे चुनते समय यह सुनिश्चित करें कि वह मज़बूत privacy features और built-in ad blockers जैसी आवश्यक सुविधाएँ प्रदान करता हो। इसके अतिरिक्त, इन web browsers में से कई extensions और add-ons का समर्थन करते हैं, और कभी-कभी ये extensions सुरक्षा जोखिम पैदा कर सकते हैं; इसलिए इन extensions को हमेशा किसी सुरक्षित ecosystem या विश्वसनीय sources से ही इंस्टॉल करें।
 
## Brave Browser Zcash Wallet Integration

Brave browser web3 ecosystem के लिए मार्ग प्रशस्त कर रहा है। कैसे? अब 2 million से अधिक unstoppable domains का उपयोग decentralized websites प्रदर्शित करने के लिए किया जा सकता है। Brave Browser की मज़बूत security features के साथ, लोग track हुए बिना internet का उपयोग कर सकते हैं। Brave और Zcash मिलकर Web3 ecosystem में नए privacy tools लाने के लिए काम कर रहे हैं। और अधिक लाभ के लिए, Brave अपने built-in crypto wallet के माध्यम से Zcash protocol के साथ तेज़ी से integrate होगा ताकि crypto transactions में क्रांतिकारी बदलाव लाया जा सके - यह integration उपयोगकर्ताओं को ZEC को बहुत आसानी से सुरक्षित रूप से store, send और receive करने में सक्षम बनाएगा, साथ ही अन्य cryptocurrencies को भी। Brave web browser और Zcash के बीच यह collaboration वास्तव में परिदृश्य बदल देगा क्योंकि उपयोगकर्ताओं को अपनी Web3 activity पर अधिक नियंत्रण मिलेगा। 

Brave browser और Zcash integration के बारे में एक और रोचक तथ्य private और decentralized messaging feature है; Brave, ECC और Filecoin Foundation की टीम Brave browser में privacy-based feature के लिए रास्ता बना रही हैं - इस collaboration का उद्देश्य Zcash protocol के माध्यम से private messages और media transmission प्रदान करना है। यहाँ, Interplanetary File System (IPFS) encrypted content के लिए secure storage capability प्रदान करने में एक आवश्यक भूमिका निभाएगा।

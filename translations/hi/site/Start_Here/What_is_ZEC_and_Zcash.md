# Zcash की मूल बातें

## संक्षेप में

- **ZEC** एक privacy-first डिजिटल currency है, जो Zcash blockchain पर निर्मित है
- Bitcoin के विपरीत, ZEC zero-knowledge proofs का उपयोग करके पूरी तरह **shielded (private) transactions** को समर्थन देता है
- आप अपनी वित्तीय गोपनीयता को नियंत्रित करते हैं: क्या प्रकट करना है, और किसे करना है, यह आप तय करते हैं
- Zcash open-source, permissionless, और decentralized है

## ZEC क्या है?

ZEC एक डिजिटल currency है जो Zcash blockchain पर आधारित है। यह वेब पर cash के सबसे निकट का रूप है। यद्यपि इसमें Bitcoin जैसी कई समान विशेषताएँ हैं, यह users को पूरी तरह private monetary transactions करने में सक्षम बनाकर Bitcoin की सबसे बड़ी कमी को दूर करता है।

#### private currency महत्वपूर्ण क्यों है?

ZEC लोगों को permissionlessly data transfer करने का अवसर देता है। peer-to-peer, permissionless money system लोगों को केंद्रीकृत संस्थाओं से स्वतंत्र होकर value store करने या दूसरों के साथ transact करने की क्षमता देता है। ZEC लोगों को यह चुनने में सक्षम बनाता है कि वे अपनी वित्तीय जानकारी दूसरों के साथ कब साझा करना चाहते हैं।

## Zcash क्या है?

Zcash "Zerocash" protocol का एक implementation है। यह एक open-source, blockchain-based ledger है, जिसमें एक उन्नत zero-knowledge proving system शामिल है। Bitcoin के code पर आधारित, यह [transaction metadata](https://nym.com/blog/what-is-metadata) की गोपनीयता बनाए रखने वाले अपने proving system के माध्यम से privacy के उच्च मानक प्रदान करने में विशेषज्ञता रखता है। अपने मूल में, Zcash private data ownership है, जिसे transactions होने पर permissionlessly transfer किया जाता है।

#### यह क्यों महत्वपूर्ण है?

Zcash Bitcoin की सबसे बड़ी कमी को हल करता है; data का private ownership और transfer। ऐसी दुनिया में जहाँ blockchain applications और cryptocurrencies अधिक व्यापक रूप से स्वीकार की जा रही हैं, pseudonymous transactions अब user privacy की रक्षा के लिए व्यवहार्य विकल्प नहीं रह गए हैं। blockchain transactions का विश्लेषण करने के लिए surveillance applications का उपयोग अधिक व्यापक होता जा रहा है।

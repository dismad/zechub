# ZecHub बनाने में मदद करें

यदि आप Zcash समुदाय के सदस्य हैं और ZecHub के निर्माण में योगदान देना चाहते हैं, तो कुछ चीज़ें हैं जो बेहद उपयोगी हैं।

न्यूज़लेटर के लिए लिंक क्यूरेट करें और उन्हें न्यूज़लेटर के GitHub पेज में जोड़ें: -> [निर्देशों के लिए यहाँ क्लिक करें](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

विकी दस्तावेज़ों की तथ्य-जांच करें, संपादन सुझाएँ, और Github में नए पृष्ठ प्रस्तावित करें

निम्नलिखित श्रेणियों में वीडियो सामग्री बनाएँ:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Zcash और Privacy ecosystem के लिए पोस्टर/ग्राफ़िक्स/एनीमेशन जैसे डिज़ाइन तैयार करें

मौजूदा विकी पृष्ठों में से किसी का भी अनुवाद करें।

बग रिपोर्ट करना

#### बाउंटी राशि के विवरण के लिए, कृपया हमारा योगदान पृष्ठ पढ़ें -> [यहाँ](https://zechub.wiki/contribute/contributing-guide#content).

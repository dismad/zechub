# Full Node ट्यूटोरियल्स

Zcash Full Nodes नेटवर्क पर transparent और shielded ट्रांज़ैक्शनों को सत्यापित करते हैं। Full node चलाकर आप नेटवर्क की लचीलापन और स्थिरता में योगदान देते हैं। इसके अतिरिक्त, Full node संचालित करने से उपयोगकर्ता को बेहतर safety और security की गारंटी मिलती है। नीचे `zcashd` या `zebrad` Full Nodes को सेट अप/अपग्रेड करने के लिए वीडियो गाइड दिए गए हैं।

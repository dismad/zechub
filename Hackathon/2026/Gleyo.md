# Gleyo — ZEC-Native Quest & Community Growth Platform

**Track:** Infrastructure

Gleyo is a Zcash-native quest and community growth platform. Any project, Zcash-based or not, can fund a community wallet in ZEC, create quests, reward contributors directly in shielded ZEC, run a built-in community chat, and track real retention — all without juggling 5 different tools.

## Live and processing real mainnet transactions

- Deposits
- Quest publishing
- Quest tasks spanning GitHub, Discord, Telegram, YouTube, quizzes, polls, puzzles, file uploads and more
- Instant ZEC rewards
- Shielded withdrawals

All working end-to-end, live on Zcash mainnet.

Since launching, Gleyo has onboarded 31+ users on mainnet, completing quests, earning shielded ZEC, and giving real feedback that's shaped the platform. Users have also completed real shielded withdrawals and a $10 community wallet has been funded and used live — not simulated flows, actual ZEC moving on mainnet.

## Built on

Self-hosted Zebra node + Nozy Wallet. Demo payments made via Zodl, recommended for its clean wallet UX, though Gleyo works with any shielded Zcash wallet.
